Wave Function Normalization and Momentum Operator

By Peter Ma | Aug 18th 2020

Previously we've introduced the statistical approach to working with physics problems. Now the issue we need to deal with is the actual wave function it self. The wave function is a distribution of probability for a given "scenario from happening" at a specific time. If this is true then by nature the wave function should have a probability of 1 if you add up all parts of that distribution. The wave function must be normalized to 1.

\[\int _{-\infty}^{+\infty} |\Psi(x,t)|^2 dx = 1\]

Normalization

Issue is can this function always be normalized to just 1? It turns out we can from this proof.

Lets take the time derivative of this function. If the time derivative of the function is 0 then it can be normalized as it must go to \(\Psi\) must go to \(\pm \infty\) or else it can't be normalized.

\[\frac{d}{dt}\int _{-\infty}^{+\infty} |\Psi(x,t)|^2 dx\]

Move the derivative under the integral

\[\int _{-\infty}^{+\infty} \frac{d}{dt}|\Psi(x,t)|^2 dx\]

We know that the function \(\Psi\) is complex given the wave equation. Thus the magnitude squared is simply the multiplication of its complex conjugate.

\[\frac{d}{dt}|\Psi(x,t)|^2 = \frac{d}{dt}(\Psi^*\Psi)\]

Product rule.

\[\frac{d}{dt}(\Psi^*\Psi) =\frac{d\Psi}{dt}\Psi^* + \frac{d\Psi^*}{dt}\Psi \]

The Schrodinger equation says...

\[i\hbar \frac{\partial \Psi}{\partial t} = -\frac{\hbar^2}{2m}\frac{\partial^2 \Psi}{\partial x^2}+ V\Psi\] \[ \frac{\partial \Psi}{\partial t} = -\frac{\hbar}{2m i}\frac{\partial^2 \Psi}{\partial x^2}+ \frac{1}{i\hbar}V\Psi\] \[ \frac{\partial \Psi}{\partial t} = -\frac{i\hbar}{2m}\frac{\partial^2 \Psi}{\partial x^2}+ \frac{i}{\hbar}V\Psi\]

The complex conjugate is just the flip the sign for all imaginary parts. Which happens to be everything.

\[ \frac{\partial \Psi^*}{\partial t} = \frac{i\hbar}{2m}\frac{\partial^2 \Psi^*}{\partial x^2}- \frac{i}{\hbar}V\Psi^*\]

We plug stuff in now.

\[\frac{d}{dt}(\Psi^*\Psi) =\frac{d\Psi}{dt}\Psi^* + \frac{d\Psi^*}{dt}\Psi \] \[\frac{d}{dt}(\Psi^*\Psi) =(-\frac{i\hbar}{2m}\frac{\partial^2 \Psi}{\partial x^2}+ \frac{i}{\hbar}V\Psi)\Psi^* + (\frac{i\hbar}{2m}\frac{\partial^2 \Psi^*}{\partial x^2}- \frac{i}{\hbar}V\Psi^*)\Psi \]

Notice that the second term \(\frac{i}{\hbar}V\Psi\) can be canceled out when you distribute the \(\Psi\)'s into the equation.

\[\frac{d}{dt}(\Psi^*\Psi) =-\Psi^*\frac{i\hbar}{2m}\frac{\partial^2 \Psi}{\partial x^2} + \Psi\frac{i\hbar}{2m}\frac{\partial^2 \Psi^*}{\partial x^2} \]

Now we can do this trick.

\[\frac{d}{dt}(\Psi^*\Psi) =\frac{\partial}{\partial x}(-\Psi^*\frac{i\hbar}{2m}\frac{\partial \Psi}{\partial x} + \Psi\frac{i\hbar}{2m}\frac{\partial \Psi^*}{\partial x}) \]

Now we can directly shove it back into the integral.

\[\int _{-\infty}^{+\infty} \frac{\partial}{\partial x}(-\Psi^*\frac{i\hbar}{2m}\frac{\partial \Psi}{\partial x} + \Psi\frac{i\hbar}{2m}\frac{\partial \Psi^*}{\partial x}) dx\] \[= -\Psi^*\frac{i\hbar}{2m}\frac{\partial \Psi}{\partial x} + \Psi\frac{i\hbar}{2m}\frac{\partial \Psi^*}{\partial x}\Big|_{-\infty}^{\infty} \]

The wave function MUST go to 0 as it tends to \(\infty\) or else it will never be normalizable.

\[= 0 \]

Therefore

\[\frac{d}{dt}\int _{-\infty}^{+\infty} |\Psi(x,t)|^2 dx = 0\]

Which suggests that the following can occur.

\[\int _{-\infty}^{+\infty} |\Psi(x,t)|^2 dx = 1\]

Normalization - Problem

Normalize the following wave function. Let \(A , \lambda, \omega, t \in \mathbb{R}\)

\[\Psi(x,t) = Ae^{-\lambda |x|} e^{-i \omega t}\]

Notice that its got an absolute value sign so to preform the integral you must split the intervals in half as such.

\[\int _{-\infty}^{+\infty} (Ae^{-2\lambda |x|} e^{-i \omega t})^2 dx = \int _{-\infty}^{0} (Ae^{2\lambda x} e^{-i \omega t})^2 dx + \int _{-\infty}^{+\infty} (Ae^{-2\lambda x} e^{-i \omega t})^2 dx\] \[ = \int _{-\infty}^{0} A^2e^{2\lambda x} e^{-2i \omega t} dx + \int _{0}^{+\infty} A^2e^{-\lambda x} e^{-2 i \omega t} dx\]

Notice that this is an event function.

\[\Psi(-x,t) = Ae^{-2 \lambda |-x|} e^{-i \omega t} =Ae^{2 \lambda |x|} e^{-i \omega t} = \Psi(x,t) \]

Thus if the integrals should be the same if they were to be an even function.

\[ = 2* \int _{0}^{+\infty} A^2e^{-2\lambda x} e^{-2 i \omega t} dx\] \[ = 2 e^{-2 i \omega t} A^2 \int _{0}^{+\infty} e^{-2\lambda x} dx\] \[ = 2 e^{-2 i \omega t} A^2 \frac{1}{-2 \lambda}[ e^{-2\lambda x}] \Big|_{0}^{\infty} \]

We know that

\[\int _{-\infty}^{+\infty} |\Psi(x,t)|^2 dx = 1\]

We can solve for \(A\)

\[1 = 2 e^{-2 i \omega t} A^2 \frac{1}{-2 \lambda}[0-1] \] \[1 = 2 e^{-2 i \omega t} A^2 \frac{1}{ 2\lambda} \] \[ e^{2 i \omega t}\lambda = A^2 \] \[ A = \sqrt{ e^{2 i \omega t}\lambda } \] \[ A = e^{ i \omega t}\sqrt{ \lambda } \]

We take out the imaginary part.

\[ A = \sqrt{ \lambda } \] \[\Psi(x,t) = \sqrt{ \lambda } e^{-\lambda |x|} e^{-i \omega t}\]

Momentum Operator

Okay so we've got this boi which we mentioned before.

\[\langle x \rangle = \int _{-\infty}^{+\infty} x |\Psi(x,t)|^2 dx \]

This helps us calculate the average expected value of \(x\) all with the same state. This doesn't mean take a particle and measure it and return it back and measure it again. That will cause a change in state. What we mean is that we have a collection of particles of same state and the average of those will be \(\langle x \rangle\).

What happens when you do this?

\[\frac{d \langle x \rangle }{d t} = ?\]

We can try and move the derivative inside.

\[\frac{d \langle x \rangle }{d t} = \frac{d }{d t} \int _{-\infty}^{+\infty} x |\Psi(x,t)|^2 dx\] \[\frac{d \langle x \rangle }{d t} = \int _{-\infty}^{+\infty} x \frac{d }{d t}|\Psi(x,t)|^2 dx\]

We expand the sqaured to a complex conjugate.

\[\Psi^2 = \Psi^*\Psi\] \[\frac{d \langle x \rangle }{d t} = \int _{-\infty}^{+\infty} x \frac{d }{d t}[\Psi^*\Psi] dx\] \[\frac{d \langle x \rangle }{d t} = \int _{-\infty}^{+\infty} x \frac{d }{d t}[-\Psi^*\frac{i\hbar}{2m}\frac{\partial \Psi}{\partial x} + \Psi\frac{i\hbar}{2m}\frac{\partial \Psi^*}{\partial x}] dx\] \[\frac{d \langle x \rangle }{d t} = \frac{-i \hbar}{2m }\int _{-\infty}^{+\infty} x \frac{d }{d t}[-\Psi^*\frac{\partial \Psi}{\partial x} + \Psi\frac{\partial \Psi^*}{\partial x}] dx\] \[\frac{d \langle x \rangle }{d t} = \frac{-i \hbar}{2m }\int _{-\infty}^{+\infty} x \frac{d }{d t}[-\Psi^*\frac{\partial \Psi}{\partial x} + \Psi\frac{\partial \Psi^*}{\partial x}] dx\]

We can do the whole integration by parts. In this form.

\[\int u dv = uv - \int v du \] Set the following, \(u = x\), \(du = dx\) and \(dv = \frac{d }{d t}[-\Psi^*\frac{\partial \Psi}{\partial x} + \Psi\frac{\partial \Psi^*}{\partial x}] \) and \(v =-\Psi^*\frac{\partial \Psi}{\partial x} + \Psi\frac{\partial \Psi^*}{\partial x}\) \[\frac{-i \hbar}{2m }\int _{-\infty}^{+\infty} x \frac{d }{d t}[-\Psi^*\frac{\partial \Psi}{\partial x} + \Psi\frac{\partial \Psi^*}{\partial x}] dx= \frac{-i \hbar}{2m } [x(-\Psi^*\frac{\partial \Psi}{\partial x} + \Psi\frac{\partial \Psi^*}{\partial x}) - \int -\Psi^*\frac{\partial \Psi}{\partial x} + \Psi\frac{\partial \Psi^*}{\partial x}dx ]\]

We know that the the ^ will be evaluated at \(\pm \infty\) and we know that \(-\Psi^*\frac{\partial \Psi}{\partial x} + \Psi\frac{\partial \Psi^*}{\partial x} = 0\) in order for the function to be normalized!! Thus we've simplified it to the following. We cancel some signs

\[= \frac{-i \hbar}{2m } \int +\Psi^*\frac{\partial \Psi}{\partial x} - \Psi\frac{\partial \Psi^*}{\partial x}dx \]

What we do then is we split the integral up.

\[= \frac{-i \hbar}{2m } \int +\Psi^*\frac{\partial \Psi}{\partial x} dx - \int\Psi\frac{\partial \Psi^*}{\partial x}dx \]

Preform integration by parts on the second integral.

\[= \frac{-i \hbar}{2m }[ \int +\Psi^*\frac{\partial \Psi}{\partial x} dx - [[\Psi^*\Psi]\Big|_{-\infty}^{\infty} - \int \Psi \frac{d \Psi^*}{dx}]]\] \[= \frac{-i \hbar}{2m } [\int +\Psi^*\frac{\partial \Psi}{\partial x} dx + \int \Psi \frac{d \Psi^*}{dx}]]\] \[\frac{d \langle x \rangle }{d t}= \frac{-i \hbar}{m } \int \Psi^*\frac{\partial \Psi}{\partial x} dx \]

What does this mean? This is the "velocity" of the expectation value of x.

\[ \langle v\rangle = \frac{-i \hbar}{m } \int \Psi^*\frac{\partial \Psi}{\partial x} dx \]

The momentum is thus... \(p = mv\)

\[ \langle p \rangle = -i \hbar \int \Psi^*\frac{\partial \Psi}{\partial x} dx \]

We can move out the derivative... and show the derivative operation.

\[ \langle p \rangle = -i \hbar \int \Psi^*\frac{\partial }{\partial x} \Psi dx \]

We call \(\frac{\partial}{\partial x}\) the momentum operator.
