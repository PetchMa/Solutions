Quantum Harmonic Oscillator Problem

By Peter Ma | Aug 21th 2020

We know from classical mechanic based on Hooke's law was definied by the following.

\[F= -kx = m \frac{d^2x}{dt^2}\]

The function for \(x(t)= A \sin(\omega t)+ B \cos(\omega t)\) and that \(\omega = \sqrt{\frac{k}{m}}\)

We also know the energy stored in the spring is the following potential...

\[V(x) = \frac{1}{2}kx^2\]

Problem: We can't use this potential as the proper potential. There is no such thing as a perfect harmonic oscillator and so we approximate it with a taylor series expansion about the minimum.

\[V(x) = (V(x_0) + V'(x_0)(x-x_0) + \frac{1}{2}V''(x_0)(x-x_0)^2+...\]

We also have \(V'(x_0) = 0\) since \(x_0\) is a minimum and the derivative must be 0. We also know that the minimum \(V(x_0)\) and we can just add on later as a constant.

Lets only take the lower order terms.

\[V(x) \approx \frac{1}{2}V''(x_0)(x-x_0)^2\]

The quantum problem for the potential can be rewritten in the following.

\[\omega = \sqrt{\frac{k}{m}}\] \[\omega^2 m = k\] \[V(x) = \frac{1}{2}m\omega^2x^2\]

We then rewrite Schrodinger equation.

\[-\frac{\hbar^2}{2m}\frac{\partial^2 \psi(x) }{\partial x^2}+ V\psi = \psi E\] \[-\frac{\hbar^2}{2m}\frac{\partial^2 \psi(x) }{\partial x^2}+ \frac{1}{2}m\omega^2x^2\psi = \psi E\]

We can see that this function is the exact same as the following with momentum operator.

\[\frac{1}{2m}[-\hbar^2\frac{\partial^2 \psi(x) }{\partial x^2}+ m^2\omega^2x^2\psi ] = \psi E\] \[\frac{1}{2m}[\hat{p}^2\psi+ m^2\omega^2x^2\psi ] = \psi E\]

We also have the following after factoring \(\psi\)

\[\frac{1}{2m}[\hat{p}^2+ m^2\omega^2x^2] = E\]

This is the Hamiltonian operator.

\[\frac{1}{2m}[\hat{p}^2+ (m\omega x)^2] = \hat{H}\]

We want to try and factor the \(\hat{H}\) as a way to solve for this. Typically this would happen

\[a^2+b^2 = (a+ib)(a-ib)\]

We can try to factor our original Hamiltonian. Problem is the Operator \(\hat{p}\) is not communtative. \(\hat{p}x \neq x\hat{p}\) but I think we can try.

\[\hat{a}_\pm = \frac{1}{\sqrt{2\hbar m \omega}}(\mp i \hat{p}+m\omega x)\]

Okay now does this recreate our original hamotonian? Lets multiply it out.

\[\hat{a}_-\hat{a}_+ = (\frac{1}{\sqrt{2\hbar m \omega}})^2(- i \hat{p}+m\omega x)( i \hat{p}+m\omega x)\] \[\hat{a}_-\hat{a}_+ = \frac{1}{2\hbar m \omega}(\hat{p}^2 + (m\omega x)^2 - i m\omega(x\hat{p} -\hat{p}x ) )\]

Okay so that didn't work out quite right... We have an extra \( x\hat{p} -\hat{p}x\)

Lets define the operation called the commutator.

\[[\hat{A}, \hat{B}] = \hat{A}\hat{B} -\hat{B}\hat{A}\]

We can use the notation in the following form.

\[\hat{a}_-\hat{a}_+ = \frac{1}{2\hbar m \omega}(\hat{p}^2 + (m\omega x)^2) -\frac{i}{2\hbar}[x, \hat{p}]\]

Okay so what the heck is the actual value of the commutator? Lets add a test function and try this out.

\[[x, \hat{p}] f(x) = [x (-i\hbar)\frac{d}{dx}f(x) - (-\hbar)\frac{d}{dx}xf(x)]\]

There's product rule in there.

\[[x, \hat{p}] f(x) = -i\hbar(x \frac{df}{dx} - x\frac{df}{dx}-f)\] \[[x, \hat{p}] f(x) = -i\hbar f(x)\]

We can drop the test function.

\[[x, \hat{p}] = -i\hbar\]

We can finally rewrite our hamiltonian as the following.

\[\hat{a}_-\hat{a}_+ = \frac{1}{\hbar \omega}\hat{H} + \frac{1}{2}\] \[\hat{H} = {\hbar \omega}(\hat{a}_-\hat{a}_+ - \frac{1}{2})\]

Notice that if we were to flip the multiplication of the factors... we get something different. They're off by a small factor.

\[\hat{a}_-\hat{a}_+ = \frac{1}{\hbar \omega}\hat{H} - \frac{1}{2}\] \[\hat{H} = {\hbar \omega}(\hat{a}_-\hat{a}_+ + \frac{1}{2})\]

If we wrote the new Schrodinger Equation we get the following.

\[\hbar \omega (\hat{a}_\pm\hat{a}_\mp \pm \frac{1}{2})\psi = E\psi\]

The trick here is you can use this to build different energies. \(\hat{a}_-\) can bring Energy down, and \(\hat{a}_+\) helps bring the energy up.

\[ \frac{1}{\sqrt{2\hbar m \omega}}(\hbar \frac{d}{dx}+ m\omega x)\psi_0 = 0\]

We rearrange and multiply it into the bracket.

\[\frac{d \psi_0}{dx} = -\frac{m\omega}{\hbar}x\psi_0\]

This makes it super easy to now solve for \(\psi_0\)

\[\int \frac{1}{\psi_0} d \psi_0= -\frac{m\omega}{\hbar}\int x dx \] \[\in(\psi_0)= -\frac{m\omega}{\hbar}x^2 \] \[\psi_0= e^{-\frac{m\omega}{\hbar}x^2} \]

We also gotta normalize this.

\[1 = \int \psi_0 ^2 dx = |A|^2 \int e^{-\frac{m\omega}{\hbar}x^2} dx\]

We get the following ...

\[\psi_0= (\frac{m \omega}{\pi \hbar})^{\frac{1}{4}}e^{-\frac{m\omega}{\hbar}x^2} \]

We can build any function \(\psi_n\) by multiplying \(\hat{a}_+^n\) with \(\psi_0\). This is the building from the ground state.

\[\psi_n = A_n \hat{a}_+^n\psi_0\]
