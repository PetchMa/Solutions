Superluminal velocities

Basically we can observe things that can appear to travel faster than the speed of light. For example, if you had a powerful lazer pointer and you moved it across a surface very far away. The point could travel faster than the speed of light only because, the point of light does not actually represent a physical thing and thus, the appearance of it traveling faster than light is just an illusion. Take the following example:

Given this geometry setup we can calculate the time required for it to reac out eyes. So at the inital emission we get

\[T_1 = {r \over c} \]

The delayed time would be the following

\[T_2 = {r - vt\cos\theta \over c}+ T1\]

We can find the APPEARANCE velocity with the simple formula

\[v_\bot = {\Delta d \over \Delta T}\] \[v_\bot = {\Delta d \over T_2 -T_1}\] \[T_2 -T_1 = {r - vt\cos\theta \over c}+ T1 - T_1\] \[T_2 -T_1 = {r - vt\cos\theta \over c}\] \[v_\bot = { vt\sin\theta \over {r - vt\cos\theta \over c}}\] \[v_\bot = { vt\sin\theta \over {r - vt\cos\theta \over c}}\] \[v_\bot = { vct\sin\theta \over r - vt\cos\theta }\]

We know that \(r = ct\)

\[v_\bot = { vct\sin\theta \over ct - vt\cos\theta }\] \[v_\bot = { v\sin\theta \over 1 - v\cos\theta/c }\]

Before we defined the velocity as a facto of the speed of light

\[\beta = v/c\] \[ v= \beta c\] \[v_\bot{\rm (apparent)} = { v t \sin \theta \over t (1 - v \cos \theta / c) }\]

With our previous equation we can replace all the velocities with \(\beta\)

\[{\beta_\bot{\rm (apparent)} = {\beta \sin \theta \over 1 - \beta \cos\theta}}\]

Now we want to find the derivative and find the MAXIMUM value of the function to show how light can appear to travel faster than the speed of light.

\[{\partial \beta_\bot{\rm (apparent)} \over \partial \theta} = 0 = { (1 - \beta\cos\theta_{\rm m}) \beta \cos \theta_{\rm m} - (\beta \sin\theta_{\rm m}) ^2 \over (1 - \beta \cos \theta_{\rm m})^2 }\]

We then solve for zero, we should get

\[\cos \theta_{\rm m} = \beta\]

The problem is we need to find a value for \(\sin \theta_{\rm m}\) to find the maximum of the function thus we use a little trick. We know that

\[\sin^2 \theta = 1- \cos^2\theta\] \[\sin \theta = \sqrt{1- \cos^2\theta}\]

In terms of cosine we can THEN replace

\[\sin \theta = \sqrt{1- \beta^2}\]

We know that this is in the form of the lortenz factor

\[\sin \theta = \gamma^{-1}\]

Plug back in.

\[{{\rm max}[\beta_\bot{\rm (apparent)}] = {\beta (1 - \beta^2)^{1/2} \over 1 - \beta^2} = \beta \gamma}\]

However if we were to consider the relativistic doppler effects we can doppler boost into the frame we know the following is true from standard relativistic doppler shift

\[\nu = {\nu' \over \gamma(1 - \beta \cos\theta)}\]

Lets set values for \(\delta \) as the following to find the DOPPLER FACTOR. How much it is doppler shifted by

\[\delta \equiv [\gamma(1 - \beta \cos \theta)]^{-1} = {\nu \over \nu'}\]

If everything is coming head on towards you you get the following as \(\theta = \pi/2\)

\[{ \delta = {\nu \over \nu'} = \gamma^{-1}}\]

For directly receding \(\theta= \pi\) sources to

\[\delta \geq (2 \gamma)^{-1}\]

for directly approaching \(\theta=0\) sources.

\[\delta \leq 2 \gamma\]
