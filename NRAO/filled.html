Two-Dimensional Aperture Antennas

Same radio telescope stuff but now its two dimensional. Thus we have the following iterated integral which is the 2-d fourier transform.

\[{f(l,m) \propto \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} g(u,v) e^{-i 2 \pi(l u + mv)} d u d v }\]

The Uniformly Illuminated Rectangular Aperture

It was derived previously that the fourier transform gave the following for a 1-d pattern. based on the similarity theorem.

\[f(l) \propto {\sin(\pi l D / \lambda) \over (\pi l D / \lambda)} \propto {\rm sinc}(l D / \lambda)\]

The two D counter part is

\[f(l,m) \propto {\rm sinc}\biggl( { l D_{\rm x} \over \lambda} \biggr) {\rm sinc} \biggl({m D_{\rm y} \over \lambda} \biggr)\]

Since the radiated power per unit area is proportional to the square of the electric field strength, the power pattern P(l) is

\[P(l) \propto {\rm sinc}^2 \biggl( {l D \over \lambda} \biggr)\]

2-d counter part is

\[P_{\rm n}(l,m) = {\rm sinc}^2\biggl({ l D_{\rm x} \over \lambda} \biggr) {\rm sinc}^2 \biggl({m D_{\rm y} \over \lambda} \biggr)\]

If we wanted to figure out the gain in any given direction we have the following. Based on energy conservation.

\[\langle G \rangle \equiv {\int_{\rm sphere} G d \Omega \over \int_{\rm sphere } d \Omega} = {4 \pi \over 4 \pi}\] \[\int G d \Omega = 4 \pi = G_0 \int_{-1}^{+1} \int_{-1}^{+1} P_{\rm n}(l,m) d l\, d m\] \[4 \pi = G_0 \int_{-1}^{+1} \Biggl[ {\sin ( \pi l D_{\rm x} / \lambda ) \over \pi l D_{\rm x} / \lambda } \Biggr]^ 2 dl \, \int_{-1}^{+1} \Biggl[ {\sin (\pi m D_{\rm y} / \lambda) \over \pi m D_{\rm y} / \lambda} \Biggr]^ 2 dm\]

If we let the variables equal => for Subsitution to solve the integral.

\[a \equiv {\pi l D_{\rm x} \over \lambda} {\rm ,~~so~~} da = { \pi D_{\rm x} \over \lambda} dl\]

Given that \(D_x >> \lambda \)

\[\int_{-1}^{+1} \Biggl[ {\sin ( \pi l D_{\rm x} / \lambda ) \over \pi l D_{\rm x} / \lambda } \Biggr]^ 2 dl \approx \biggl[ \int_{-\infty}^\infty {\sin^2 a \over a^2} da \biggl] \,{\lambda \over \pi D_{\rm x}} = {\lambda \over D_{\rm x}}\]

We use a table for the integral.

\[G_0 = {4 \pi D_{\rm x} D_{\rm y} \over \lambda^2}\]

Therefore the gain we calculated is

\[{G = {4 \pi D_{\rm x} D_{\rm y} \over \lambda^2} {\rm sinc}^2\biggl({ l D_{\rm x} \over \lambda} \biggr) {\rm sinc}^2 \biggl({m D_{\rm y} \over \lambda} \biggr)} \]

We can approximate with \(\theta < 15deg\)

\[{\approx {4 \pi D_{\rm x} D_{\rm y} \over \lambda^2} {\rm sinc}^2\biggl({ \theta_{\rm x} D_{\rm x} \over \lambda} \biggr) {\rm sinc}^2 \biggl({\theta_{\rm y} D_{\rm y} \over \lambda} \biggr)}\]

We find that the on-axis effective collecting area is

\[{\rm max}(A_{\rm e}) = {\lambda^2 G_0 \over 4 \pi} = {4 \pi \lambda^2 D_{\rm x} D_{\rm y} \over 4 \pi \lambda^2} = D_{\rm x} D_{\rm y} = A_{\rm geom}\]

The Circular Gaussian Illumination Pattern

A good approximation for "real" radio telescopes with practical feeds and circular apertures is a circular Gaussian illumination pattern. We have the guassian function

\[g(u) \equiv {1 \over \sqrt{2 \pi} \sigma} \exp \biggl( - {u^2 \over 2 \sigma^2} \biggr)\] \int_{-\infty}^{\infty} g(u) d u = 1

For circular illumination problem we have the following

\[f(l,m) \propto \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} \exp \biggl( - { u^2 \over 2 \sigma^2} \biggr) \exp \biggl( - { v^2 \over 2 \sigma^2} \biggr) \exp [ -i 2 \pi (l u + m v)] d u d v~\]

Notice we can split them up into the components of each direction

\[f(l) \equiv\int_{-\infty}^\infty \exp \biggl( - {u^2 \over 2 \sigma^2}\biggr) \exp ( - i 2 \pi l u ) du\] \[f(m) \equiv \int_{-\infty}^\inftyw \exp \biggl( - {v^2 \over 2 \sigma^2}\biggr) \exp ( - i 2 \pi m v ) dv\]

Notice that each of these are just one dimensional fourier transforms of the guassian!! In each directions woahhhh! We know the fourier transform of a guassian as this \(F(s) = \exp (- \pi s^2)\)

\[f(l) \propto \exp( - \pi l^2 \times 2 \pi \sigma^2)\] \[f(m) \propto \exp( - \pi m^2 \times 2 \pi \sigma^2)\]

In the end we get

\[f(l,m) \propto \exp[- 2 \pi^2 \sigma^2 (l^2 + m^2)]\]
