The Rayleigh-Jeans Law

If we were to rearrange the first equation for the wave lengths

\[n_{\rm x} = {2 a \over \lambda_{\rm x}}, n_{\rm y} = {2 a \over \lambda_{\rm y}}, n_{\rm z} = {2 a \over \lambda_{\rm z}}\]

If we substitude the original in for \(\lambda_x\)... etc

\[n_{\rm x} = {2 a \cos\,\alpha \over \lambda}, n_{\rm y} = {2 a \cos\,\beta \over \lambda}, n _{\rm z} = {2 a \cos\,\gamma \over \lambda}\]

The solution was to quantize the energy into discrete partitions. Energy existed as a individual packet of energy and not infact continous. We are given the Planck Constant.

\[E = n h \nu, \qquad n = 1,~2,~3,~...\]

We were given this probability distribution by boltzmann saying that

\[P(E) \propto \exp\biggl({-E \over k T}\biggr)\]

Subbing in our new found form of energy we get

\[P(E) = P(n h \nu) \propto \exp\biggl({ - n h \nu \over k T}\biggr)\]

the average energy per mode is calculated by summing over only the discrete energies permitted instead of integrating over all energies

\[\langle E \rangle = {\sum_{n = 0}^\infty n h \nu P(n h \nu) \over \sum_{n = 0}^\infty P(n h \nu)} = {\sum_{n = 0}^\infty n h \nu \exp\bigl({ - n h \nu \over k T}\bigr) \over \sum_{n = 0}^\infty \exp\bigl({ - n h \nu \over k T}\bigr)}\]

To solve the following we can do this. let \({1 \over k T} = a\)

\[\langle E \rangle = {\sum_{n = 0}^\infty n h \nu \exp\bigl({ -a n h \nu}\bigr) \over \sum_{n = 0}^\infty \exp\bigl({ -a n h \nu}\bigr)}\]

Now to help simplify, take the ln and the derivative in respect to a of the following.

\[- {d \over d\alpha}\biggl[\ln \sum_{n = 0}^\infty \exp(- \alpha n h \nu)\biggr]\]

Eval the derivative with chain rule

\[- {d \over d\alpha}\biggl[\ln \sum_{n = 0}^\infty \exp(- \alpha n h \nu)\biggr] = {1 \over \sum_{n = 0}^\infty \exp(- \alpha n h \nu)} {-d \over d \alpha} \biggl[\sum_{n = 0}^\infty \exp(-\alpha n h \nu)\biggr]\] \[- {d \over d\alpha}\biggl[\ln \sum_{n = 0}^\infty \exp(- \alpha n h \nu)\biggr] = {1 \over \sum_{n = 0}^\infty \exp(- \alpha n h \nu)} \biggl[\sum_{n = 0}^\infty {-d \over d \alpha}\exp(-\alpha n h \nu)\biggr]\] \[- {d \over d\alpha}\biggl[\ln \sum_{n = 0}^\infty \exp(- \alpha n h \nu)\biggr] = {1 \over \sum_{n = 0}^\infty \exp(- \alpha n h \nu)} \biggl[\sum_{n = 0}^\infty \alpha n h \exp(-\alpha n h \nu)\biggr]\] \[- {d \over d\alpha}\biggl[\ln \sum_{n = 0}^\infty \exp(- \alpha n h \nu)\biggr] = {\sum_{n = 0}^\infty n h \nu \exp (- \alpha n h \nu) \over \sum_{n = 0}^\infty \exp(- \alpha n h \nu)}\]

Now all we need to evaluate is the following:

\[\sum_{n = 0}^\infty \exp(- \alpha n h \nu) = 1 + [\exp( - \alpha h \nu)]^1 + [\exp( - \alpha h \nu)]^2 + \dots ~.\]

We notice its like a geometric series.

\[\sum^{\infty} ax^n = \frac{a}{1-x}\]

Therefore the following is true.

\[\sum_{n = 0}^\infty \exp(- \alpha n h \nu) = \frac{1}{1- \exp(- \alpha n h \nu)}\]

\[- {d \over d\alpha}\biggl[\ln \frac{1}{1- \exp(- \alpha n h \nu)}] = {\sum_{n = 0}^\infty n h \nu \exp (- \alpha n h \nu) \over \sum_{n = 0}^\infty \exp(- \alpha n h \nu)}\]
