Electrodynamics Solving For Potential

The key to understand is, solving for electric field is hard. The easy part is if you have the potential. However even the potential can be hard to solve. What we know is the following is true about the potential.

\[\nabla ^2 V = - \frac{\rho}{ \epsilon_0}\]

We also know that since \(\rho=0\) everywhere where there is no charge density.

\[\nabla ^2 V = 0\]

This requires us to solve a second order partial differential equation. Which is hard. So what we do is that we can think of it in another way.

Laplace's Equation in One Dimension

If we let V depend on just x we get

\[\frac{d^2 V}{dx^2} = 0\]

The general solution would just be

\[V(x) = mx+b\]

We can also reqrite it as the average between \(+a\) and \(-a\)

\[V(x) = \frac{1}{2}(V(x+a)+v(x-a))\]

Laplace's Equation in Two Dimension

We can do the same in 2 dimensions. Except what we do is we average the radius around the the surface.

\[\frac{\partial^2 V}{\partial x^2} +\frac{\partial^2 V}{\partial y^2} = 0\]

We can pretend to wrap around the area a thin sheet of film and average around the circle to get the value of the center.

\[V(x,y) = \frac{1}{2\pi R} \oint V dl\]

Laplace's Equation in Three Dimension

We can then find value at the center of a solid by taking the average of the surface area of the sphere around it given the following...

\[V(r) = \frac{1}{4 \pi R^2} \oint V da\]

Uniqueness Theorem

All that was important in this section about the uniqueness theorem is that as long as you can find one solution to the laplace equation, it IS the solution to the laplace equation.

Classic Image Problem

Suppose a point charge of q is held a distance d above a conducting plane with a potential of V=0. How can we find the potential?

The problem here is that the conducting plane will have an induced charge creating a surface charge density that we don't know thus making the problem hard to solve. What we do instead is we take the problem and flip the geometry by positioning another opposite charge to the original creating a potential of 0.

Imagine the same problem, but remove the plane and place an equal but negative charge opposite to the first charge. This is the same problem but the potential is easier to solve for. Simply add the to charges together. Obviously this configuration can get a lot harder all we need to know is to configure it in a way to balance out a total potential of 0 given the boundary conditions.

\[V(x,y,z) = \frac{1}{4\pi \epsilon_0} [\frac{q}{\sqrt{x^2+y^2+(z-d)^2}}-\frac{q}{\sqrt{x^2+y^2+(z+d)^2}}]\]

We just arranged it so that the point in the middle will experience a potential of 0. Now we can solve for it.

How would we find the charge distribution on the plane? The charge induced will follow a charge distribution of the following

\[\sigma = -\epsilon_0 \frac{\partial V}{\partial n}\]

Where \(n\) is the normal to the surface of the conductor.

Solving the Partial Second Order Differential Equation - Seperation of Variables technique

We want to reduce Laplace's equation into something manageable. We know the following.

\[\nabla ^2 V = 0\] \[\frac{\partial^2 V}{\partial x^2} +\frac{\partial^2 V}{\partial y^2} = 0\]

But lets say we can seperate this into independent functions such as

\[V(x,y) = W(x)U(y)\]

Then by differenting the values we get the following

\[U\frac{d^2 W}{dx^2} + W\frac{d^2 U}{dy^2} = 0\]

We divide by \(U\) and \(W\).

\[\frac{1}{W}\frac{d^2 W}{dx^2} + \frac{1}{U}\frac{d^2 U}{dy^2} = 0\]

Now since both components don't depend on each other we can set a seperation variable

\[\frac{1}{W}\frac{d^2 W}{dx^2} = C_1\] \[\frac{1}{U}\frac{d^2 U}{dy^2} = C_2\] \[C_1 + C_2= 0\]

RECALL: That we typically solve second order differential equations by solving for the roots of the second order differential.

\[ay'' + by'+cy = 0\] \[ak^2+ bk+c = 0\]

Depending on if \(k\) is complex we'd use different solution formulas.

In this case we have a complex solution since

\[C_1 =-C_2\]

Where the following is true

\[\frac{d^2 W}{dx^2} = k^2 W\] \[\frac{d^2 U}{dy^2} = -k^2 U\]

For complex solutions of \(k\) we get the following equations to solve

\[W(x) =Ae^{kx}+Be^{-kx}\] \[U(y) =C\sin(ky)+ D \cos(ky) \] \[V(X,Y) =(Ae^{kx}+Be^{-kx})(C\sin(ky)+ D \cos(ky)) \]

Now we need to consider the boundary conditions to solve a particular potential problem. Given the seperation of variables, this becomes easy to solve.

We know that \(D =0 \) because V=0 when y=0. \(D \cos(K(0))!=0\)

We know that \(A =0 \) because V=0 when \(x \to \infty\). \(Ae^{kx}!\to 0\)

\[V(X,Y) =(Be^{-kx})(C\sin(ky)) \]

We also know that when the potential approaches a surface \(y \to a\) we also get a boundary condition of 0.

\[0 =C\sin(ky) \] \[0 =C\sin(ka) \] \[k= \frac{n\pi}{a}\]

Where n is integer number

\[V(X,Y) =C_ne^{-\frac{n\pi}{a}x}\sin(\frac{n\pi}{a}y) \]

Because of this we get an infinite number of solutions as long as this is satisfied with an integer value for n.

\[V(X,Y) =\sum_{n=0}^{\infty}C_ne^{-\frac{n\pi}{a}x}\sin(\frac{n\pi}{a}y) \]

Now we need to solve for the value of \(C_n\). We use something called a fourier trick! Its really neat. Basically we reduce the problem down by canceling out some terms..... So we know this is true.

\[V(0,Y) =\sum_{n=0}^{\infty}C_n\sin(\frac{n\pi}{a}y) \] \[V(0,Y) =V_0 \]

The fourier trick is where we integrate both sides which contains a value \(n`\)

\[V(0,Y) =\int \sum_{n=0}^{\infty}C_n\sin(\frac{n\pi}{a}y)sin(\frac{n`\pi}{a}y) dy \] \[\sum_{n=0}^{\infty} C_n\int_0^a \sin(\frac{n\pi}{a}y)sin(\frac{n`\pi}{a}y) dy = \int_0^a V_0 \sin(\frac{n`\pi}{a}y) dy \]

Nothing has changed just yet... if we integrated the left hand side we get the following

\[\int_0 ^a \sin(\frac{n\pi}{a}y)sin(\frac{n`\pi}{a}y) dy =0\]

All except when \(n=n`\) where

\[\int_0 ^a \sin(\frac{n\pi}{a}y)sin(\frac{n`\pi}{a}y) dy =\frac{a}{2}\]

THUS the entire sumation gets killed off because every value of \(n\) will equal to 0 except when it equals the \(n`\). So \(n=n`\)

\[\frac{a}{2}C_n = \int_0^a V_0 \sin(\frac{n\pi}{a}y) dy \]

This we can solve...

\[C_n = \frac{2}{a}\int_0^a V_0 sin(\frac{n\pi}{a}y) dy \] \[C_n = \frac{2V_0}{n\pi} (1- \cos(n\pi)) \]

The \(1- \cos(n\pi)\) value will be determined if its even or odd. if \(n=even\) then it gets zeroed out, else it will return 1. Therefore with the final conclusion

\[C_n = \frac{2V_0}{n\pi} \] \[V(X,Y) =\sum_{n=1,3,5...}^{\infty}\frac{2V_0}{n\pi}e^{-\frac{n\pi}{a}x}\sin(\frac{n\pi}{a}y) \]
