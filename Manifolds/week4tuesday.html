Recall that a function for \(f:\mathbb{R}^n \to \mathbb{R}\) we proved that \(f\) is continous at \(a\) if and only if each \(f_i\) is continous at \(a\) where \(f(x) = [f_1(x)\cdots f_k(x)]\)

The limits for each componenet the limit must approach to the value then it implies that the limit will be the same for the overall.

Theorem

If \(A\subseteq \mathbb{R}^n\) a subspace and \(f:A \to \mathbb{R}^k\) then \(f\) is differentiable at at \(a \in A \Leftrightarrow f_i\) differentiable at \(a\).

Proof: \(f\) is differentiable at \(a \Leftrightarrow \exists B\) a \(k\times n \) matrix with \[\lim_{h\to 0} \frac{f(a+h) - f(a) -Bh}{|h|}=0\]

Given a \(k\times n\) matrix \(B\) define that \(F^B(hh) = \frac{f(a+h)-f(a) -Bh}{|h|}\)

Note that each of the \(F_i^B(h) = \frac{f_i(a+h)-f_i(h)-B_ih}{|h|}=0\)

We will define some new notation to help us with this big theorem. Class of functions, we say that \(f:A\to\mathbb{R}^k\) is of class \(C^1\) on \(A\) if for each \(i\in \{1\cdots k\}\) and \(j\in \{1\cdot n\}\) \(D_j f_i\) exists and is continous on \(A\).

\(f:A\subseteq \mathbb{R}^n \to \mathbb{R}^k\) then \(f_i: A \subseteq \mathbb{R^n}\to \mathbb{R}\) where that \(Df_i(a)\) is a \(1\times n\) matrix where it is a point. We can think of \(D_j f_i\) as a map from \(A\to \mathbb{R}\).

We cannot think of the entire thing as a function anymore, each individual compoenent gives rise to a function. This is because the ENTIRE derivative gives out a MATRIX instead now. Example:

\[f(x,y) = (x^2y,x-y,3xy)\]

For each of these we can compute the following:

\[f_1(x,y) = x^2y\] \[f_2(x,y) = x-y\] \[f_3(x,y) = 3xy\]

Thus we get the following:

\[Df(x,y) = \left[ \begin{matrix} 2xy & x^2\\ 1 & 0-1 \\ 3y &3x \\ \end{matrix}\right] \]

\(Df\) maps to \(\mathbb{R}^2\) and maps to a matrix.

If each of the individual functions are existent and continous then it is class \(C^1\).

We say that something is of class \(C^r\) on \(A\) if \(D_jf_i: A\to \mathbb{R}\) then it is of class \(C^{r-1}\).

We say that things are \(C^\infty\) if it is infinitly differentiable.
