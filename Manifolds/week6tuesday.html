Inverse Function Theorem

Observe that if \(a\subseteq \mathbb{R}^n\) is a subspaces and that \(f: A \to \mathbb{R}^n\) such that there exists \(U \subseteq A\) open with the restriction \(F|_U \) is injective, then \(f|_U: U \to f(U)\) is bijective. If \(g = {f|_U}^{-1}: f(U) = \to U\) is differentiable then \(D(g\circ f) = D\iota = I\) Also, \(D(g\circ f) (a) = Dg(f(a))Df(a)\).

Thus we have that \(Dg(f(a)) = (Df(a))^{-1}\).
