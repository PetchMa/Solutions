Metric

  1. Function is definite: \(d(y,x) = 0 \) if and only if \(x=y\)
  2. Function is symmetrical: \(d(y,x) = d(x,y) \)
  3. Obeys triangle inequality commutes: \(d(y,x) \leq d(x,z)+ d(z,y) \)

We don't want to introduce more structure then it is needed and so we do not want to simply use the inner product to define a norm. The inner product helps with angles and such, so we stop using inner products as metrics.

Sometimes proofs are easier with one metric over another.

Def: \(\epsilon -\)Neighbourhoods

We denote such as: \(U(x_0;\epsilon) = \{x\in \mathbb{R}^n: d(x,x_0) < \epsilon\}\)

Given the Euclidean metric we get a ball, if we use the \(\text{sup}\) metric we have a cube.

These intervals in analysis in 1 dimensional space are now described by these \(\epsilon -\)Neighbourhoods

Def: Open Set

A set \(U \subseteq \mathbb{R}^n\) is considered open if \(\forall x \in U, \exists \epsilon>0\) such that \(U(x; \epsilon) \subset U\). \(U(x,\epsilon)\) is also open.

We use these to measure the closeness.

Def: Closed set

\(A\) is closed if \(\mathbb{R}^n - A\) is open.

Theorem

Suppose that \(Y\subseteq \mathbb{R}^n\) a subspace. Then \(U\subseteq Y\) is open in \(Y \Leftrightarrow U = U'\cap Y\) where \(U'\) is open.

The idea here is we add "cushioning" for the set \(U\) to wiggle around.

We want to define the closed set in a better way:

Def: Limit Point

We say that \(x\in \mathbb{R}^n\) is a limit point of \(X\) if for all \(\epsilon > 0 \) then every deleted neighbourhood \(U(x,\epsilon) - \{x\} \cap X \neq 0\).

Def: Closure

The closure of \(X\) is the union of its limit points. Denoted by \(\overline X\)

Theorem

Closure are closed. More specifically \(X\) is closed if it is equal to its \(\overline X\).

Closed sets are closed under limits.

Limits and Continuity

We can recall in the 1-D case that the limit \(\lim_{x\to x_0} f(x) = y\) if by the \(\delta - \epsilon\) definition.

We can generalize this, we replace these with a metric:

\[\lim_{x\to x_0} f(x) = y \Leftrightarrow \forall \epsilon >0 \exists \delta > 0 : 0< d(x,x_0) < \delta \text{ then }d(f(x),y) < \epsilon\]

This is one way to define the limit. We want to use the power of our sets and specifically the \(\epsilon-\) neighbourhood. \[\forall \epsilon >0 \exists \delta > 0 : \underbrace{0< d(x,x_0)< \delta}_{\text{deleted delta neigh}} \text{ then } \underbrace{d(f(x),y) < \epsilon}_{epsilon neigh}\]

Definition: Limit [without \(\epsilon-\delta\)]

Suppose that \(X \subseteq \mathbb{R}^n , Y \subseteq \mathbb{R}^k \) subspaces and \(f:X\to Y\) we say that \(\lim_{x\to x_0} f(x) = y\) if and only if \(\forall V \subseteq Y\) is open in \(Y\) containing \(y\) that \(\exists U \subseteq X\) open in \(X\) containing \(x_0\) such that \(f(U-\{x_0\}) \subseteq V\)

For any openset we can map it with \(f\) and it will fit inside of the open set.

Ex) \(\forall x \in \mathbb{R}^n, |x|\leq ||x| \leq \sqrt{n}|x|\).

Continuity [without \(\epsilon-\delta\)]

\(f: X \to Y\) is continous at \(x_0\in X \Leftrightarrow \forall V \subseteq Y\) open in \(Y\) containing \(f(x_0)\) that there \(\exists U \subseteq X\) open in \(X\) containing \(x_0\) such that \(f(U) \subseteq V\).

Theorem

  1. Composition of functions tha are continous are also continous.
  2. Sum and Differences are continous if it is continous.
  3. Products and ratios are continous if it is continous.
  4. Constant function is continous
  5. \(\Pi_i: \mathbb{R}^k \to \mathbb{R}\) is a projection such that: \(\Pi_i(x_1 \cdots x_i \cdots) = x_i\). Prove using the sup metric.
