Problem 1.8

Suppose that \(z \neq âˆ’1\) is a complex number of unit magnitude: \(|z| = 1\). Show that \(w =\frac{ 1+z}{|1+z|}\) is a square root of z.

Solution

If it is true that \(w = \sqrt{z}\) \( \Rightarrow |w^2| = |z|\).

We know that we can write \(|w^2| = w\cdot\bar{w}\) where \(w,\bar{w}\in \mathbb{C}\).

Since \(|z|=1\) and \(|w^2| = |z|\). Then If \(w\cdot\bar{w} = 1 \Rightarrow w = \sqrt{z}\)

We know that \(w =\frac{ 1+z}{|1+z|}\). Let \(z = a+bi\).

\[w= \frac{ 1+a+bi}{|1+a+bi|}\]

We can plug it back in and get...

\[\bar{w}= \frac{ (1+a)-bi}{|(1+a)-bi|}\]

Sub back into original equation

\[|w^2| = w\cdot\bar{w}\] \[|w^2| = \frac{ (1+a)+bi}{|(1+a)+bi|}\cdot\frac{ (1+a)-bi}{|(1+a)-bi|}\] \[|w^2| = \frac{ (1+a)^2+b^2}{|(1+a)^2+b^2|}\] \[|w^2| = \frac{ a^2+2a+1+b^2}{|a^2+2a+1+b^2|}\]

We know that \(|z|=1\), since \(z = a+bi\) then, \(a^2+b^2=1\), we can sub in the equation

\[|w^2| = \frac{ 2+2a}{|2+2a|}\]

Since there's no more components we can drop the magnitude bars

\[|w^2| = \frac{ 2+2a}{2+2a}\] \[|w^2| = 1\] \[|w|^2 = 1\] \[\therefore |w|^2 = |z|\]

Since \(|w|^2=w \cdot \bar{w}\) gives \(|z|\) and \(|w| = |\bar{w}|\), then we shown that \(|w|\) is the magnitude for \(|\sqrt{z}|\).

However, \(|w^2| = |z|\) \( \nRightarrow w = \sqrt{z}\) :((. We haven't proved it yet.

But we also know that since \(w\) is unique such that the complex conjugate gives rise to it's magnitude square. We can say that \(w\) or \(\bar{w}\) is equal to\(\sqrt{z}\) since no other value can give rise to magnitude square with the same behaviour. A perfect square will only ever have 2 unique roots.
