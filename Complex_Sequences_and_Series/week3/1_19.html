Part D¶

Graph the line $x = 2y$ with the $h(x + iy) = e^x(\cos(y) + i \sin(y))$

First I create the real axis that are the inputs. With the get_input_function_2 where $x=2y$

Then I create the function get_output_function $h(x + iy) = e^{x+iy}$ which is the same as $e^x(\cos(y) + i \sin(y))$

A hypothesis was that this will scale the spiral found in part c. Lets first rewrite the equation in part C We can sub in values and simplify this expression. $$h(x + iy) = e^x(\cos(y) + i \sin(y))$$ $$h(x + iy) = e^x(\cos(x) + i \sin(x))$$ $$h(x + iy) = e^x(e^{ix})$$ $$h(x + iy) = e^{x+ix}$$ Now we can simplify our new part D expression $$h(x + iy) = e^x(\cos(y) + i \sin(y))$$ Sub in values $$h(x + i2x) = e^x(\cos(2x) + i \sin(2x))$$ Sub with identities $$h(x + i2x) = e^x(\cos^2(x)-\sin^2(x) + i 2\sin(x)\cos(x))$$ This can be factored $$h(x + i2x) = e^x(\cos(x)-i\sin(x))^2$$ $$h(x + i2x) = e^x(e^{ix})^2$$ $$h(x + i2x) = e^x(e^{2ix})$$ By repeating the part we did before, there is infact a scaling factor at play. It appears that the "rotation" created by $e^{2ix}$ has half the period of the original $e^{ix}$ which suggests that the graph would be more "tightly" wounded.

Part E¶

Graph the line $x = -y$ with the $h(x + iy) = e^x(\cos(y) + i \sin(y))$

First I create the real axis that are the inputs. With the get_input_function_3 where $x=-y$

Then I create the function get_output_function $h(x + iy) = e^{x+iy}$ which is the same as $e^x(\cos(y) + i \sin(y))$

If we were to simplify our expression. $$h(x + iy) = e^x(\cos(y) + i \sin(y))$$ $$h(x + iy) = e^x(\cos(-x) + i \sin(-x))$$ $$h(x + iy) = e^x(\cos(x) - i \sin(-x))$$ Notice its just the complex conjugate $$h(x + iy) = e^x(e^{-ix})$$ $$h(x + iy) = (e^{x-ix})$$ It appears to experience a reflection in the imaginary component. When switching the sign of the vertical component its typically a reflection along the horizontal.
