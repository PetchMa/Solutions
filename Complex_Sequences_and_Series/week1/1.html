Problem 1.6

Is it true that \(Re(zw) = Re(z) Re(w)\) for all complex numbers z and w? If so, show that it is always true. If not, give an example of complex numbers z and w such that \(Re(zw) \neq Re(z) Re(w)\).

Solution - Proof by Contradiction.

Assume that \(Re(zw) = Re(z) Re(w)\) is true.

Let \(z = a+bi\), \(w = c+di\) and \(y = Re(zw) - Re(z) Re(w)\)

If \(Re(zw) = Re(z) Re(w)\), then \(y=0\)

\[y = Re(zw) - Re(z) Re(w)\] \[y= Re((a+bi)(c+di)) - Re( a+bi) Re(c+di)\] \[y= Re((ac+adi-bd+cbi)) - Re( a+bi) Re(c+di)\] \[y= Re((ac-bd)+(ad+cb)i) - Re( a+bi) Re(c+di)\] \[y= (ac-bd) - (ac)\] \[y= -bd\]

Since z and w are complex numbers, \(b, d \neq 0 \) therefore, \(y \neq 0\).

Since \(y \neq 0\) contradicts the assumption that \(Re(zw) = Re(z) Re(w)\), \(\forall z,w \in \mathbb{C} \) therefore \(Re(zw) \neq Re(z) Re(w)\), \(\forall z,w \in \mathbb{C} \)

Problem 1.7

This problem provides an algebraic proof of the triangle inequality, Proposition 1.7. Let z and w be complex numbers.

(a) Show that \(|z+w|^2 =|z|^2 + z\bar{w} + w\bar{z} + |w|^2 \)

Let \(z = a+bi\), \(w = c+di\)

\(RS =|z|^2 + z\bar{w} + w\bar{z} + |w|^2 \)

\(RS =|a+bi|^2 + (a+bi)(c-di) + (c+di)(a-bi) + |c+di|^2 \)

\(RS =(a^2+b^2)+ (a+bi)(c-di) + (c+di)(a-bi) + (c^2+d^2) \)

\(RS =(a^2+b^2)+ (ac-adi+cbi+bd) + (ac+adi-cdi+bd) + (c^2+d^2) \)

\(RS =(a^2+b^2)+ (ac+bd) + (ac+bd) + (c^2+d^2) \)

\(RS =(a^2+b^2)+ 2(ac+bd) + (c^2+d^2) \)

On the left hand side.

\(LS=|z+w|^2\)

\(LS=|(a+bi) + (c+di)|^2\)

\(LS=|(a+c) + (b+d)i|^2\)

\(LS=(a+c)^2 + (b+d)^2\)

\(LS=a^2+2ac+c^2 + b^2+2bd+d^2\)

\(LS=a^2+c^2+2(ac+bd) + b^2+d^2\)

\[LS=RS\] \[\therefore True\]

(b) Justify the following chain of equalities and inequalities: \(z\bar{w} + w\bar{z} = z\bar{w} + \bar{z\bar{w}} = 2Re(z\bar{w}) \leq 2 |z||w| \)

First Equality

If \(z\bar{w} + w\bar{z} = z\bar{w} + \bar{z\bar{w}}\) then \((z\bar{w} + w\bar{z}) - (z\bar{w} + \bar{z\bar{w}}) =0\)

Let \(z = a+bi\), \(w = c+di\) and \(y = (z\bar{w} + w\bar{z}) - (z\bar{w} + \bar{z\bar{w}}) \)

\(y = (z\bar{w} + w\bar{z}) - (z\bar{w} + \bar{z\bar{w}}) \) \(y = ((a+bi)(c-di) + (c+di)(a-bi)) - ((a+bi)(c-di) + \bar{(a+bi)(c-di)}) \) \(y = ((ac-adi+cbi+bd) + (ac+adi-cdi+bd)) - ((ac-adi+cbi+bd) + \bar{(ac-adi+cbi+bd)}) \) \(y = ( 2(ac+bd)) - ((ac-adi+cbi+bd) + (ac+adi-cbi+bd)) \) \(y = ( 2(ac+bd)) - (2(ac+bd)) \) \[ y = 0 \]

Since \(y=0\) it proves that \(z\bar{w} + w\bar{z} = z\bar{w} + \bar{z\bar{w}}\)

Second Equality

If \(z\bar{w} + w\bar{z} = 2Re(z\bar{w} )\) then \((z\bar{w} + w\bar{z}) - Re(z\bar{w}=0\)

Let \(z = a+bi\), \(w = c+di\) and \(y = (z\bar{w} + w\bar{z}) - (2Re(z\bar{w}) \)

\(y = ((a+bi)(c-di) + (c+di)(a-bi)) - (2Re((a+bi)(c-di)) \) \(y = ((ac-adi+cbi+bd) + (ac+adi-cdi+bd)) - (2Re((ac-adi+cbi+bd) ) \) \(y = ((ac+bd) + (ac+bd)) - 2(ac+bd) \) \[ y = 0 \]

Since \(y=0\) it proves that \(z\bar{w} + w\bar{z} = 2Re(z\bar{w})\)

Inequality

\(2Re(z\bar{w}) \leq 2 |z||w|\)

Case \(2Re(z\bar{w}) = 2 |z||w|\)

Let \(z = a+bi\), \(w = c+di\) and \(y = ( 2 |z||w|) - (2Re(z\bar{w}) ) \)

\(y = ( 2 |a+bi||c+di|) - (2(ac+bd) ) \) \(y = ( 2 \sqrt{a^2+b^2} \sqrt{c^2+d^2}) - (2(ac+bd) ) \) \(y = ( 2 \sqrt{a^2c^2+b^2c^2+a^2d^2 +b^2d^2} ) - (2(ac+bd) ) \)

Since \(\exists z=w, \Rightarrow\) \(\exists a=c\) and \(\exists b=d\)

\(y = ( 2 \sqrt{a^4+b^2a^2+a^2b^2 +b^4} ) - (2(a^2+b^2) ) \) \(y = ( 2 \sqrt{(a^2+b^2)^2} ) - (2(a^2+b^2) ) \)

\(y = ( 2 (a^2+b^2) ) - (2(a^2+b^2) ) \) \[y = 0 \] \[\therefore \exists 2Re(z\bar{w}) = 2 |z||w|\]

Case \(2Re(z\bar{w}) < 2 |z||w|\), \(Re(z\bar{w}) > 0\)

If \(2Re(z\bar{w}) < 2 |z||w|\), then \( [(2 |z||w|)^2 - (2Re(z\bar{w}))^2] >0\) assuming that \(Re(z\bar{w}) > 0\)

Let \(z = a+bi\), \(w = c+di\) and \(y = (2 |z||w|)^2 - (2Re(z\bar{w}))^2 ) \)

\(y = (2 |z||w|)^2 - (2Re(z\bar{w}))^2 \)

\(y = (4 (a^2c^2+b^2c^2+a^2d^2 +b^2d^2)) - (4(ac+bd)^2) ) \) \(y = 4 (a^2c^2+b^2c^2+a^2d^2 +b^2d^2) - 4(a^2c^2+2acdb +b^2d^2) \) \(y = 4b^2c^2+a^2d^2 -8acdb \)

\(y = 4(b^2c^2+a^2d^2 -2acdb) \)

\(y = 4(bc-ad)^2 \)

Since this is \(y\geq 0\) when \(Re(z\bar{w}) > 0\) the inequality is true for this case.

Case \(2Re(z\bar{w}) < 2 |z||w|\), \(Re(z\bar{w}) < 0\)

We just need to subtract by the negative of the magnitude.

If \(2Re(z\bar{w}) < 2 |z||w|\), then \( [(2 |z||w|)^2 + (2Re(z\bar{w}))^2] >0\) assuming that \(Re(z\bar{w}) < 0\)

Let \(z = a+bi\), \(w = c+di\) and \(y = (2 |z||w|)^2 + (2Re(z\bar{w}))^2 ) \)

\(y = (4 (a^2c^2+b^2c^2+a^2d^2 +b^2d^2)) + (4(ac+bd)^2) ) \)

All the terms will be positive thus, \(y \geq 0\)

Case \(Re(z\bar{w}) = 0\)

\(Re(z\bar{w}) = 0 \Rightarrow\) \(bd =- ac\)

We can burrow the results from previous cases.

\(y = (4 (a^2c^2+b^2c^2+a^2d^2 +b^2d^2)) + (4(ac+bd)^2) ) \) \(y = (4 (a^2c^2+b^2c^2+a^2d^2 +(-ac)^2)) ) \)

Since all terms are positive, we can conclude that \(y \geq 0\) when \(Re(z\bar{w}) = 0\)

\(\therefore \) we can say that \(z\bar{w} + w\bar{z} = z\bar{w} + \bar{z\bar{w}} = 2Re(z\bar{w}) \leq 2 |z||w| \) is true.

(c) Combine parts (a) and (b) to show that \(|z + w|^2 \leq (|z|+|w|)^2\). Then take the square root of both sides to obtain the triangle inequality.

Let \(z = a+bi\), \(w = c+di\) and \(y = (|z|+|w|)^2 - |z + w|^2 \)

If \(y \geq 0\) then the inequality is true.

\(y = (|z|+|w|)^2 - |z + w|^2\)

\(y = (|z|^2+2|z||w|+|w|^2) - |z + w|^2\)

\(y = ((a^2+b^2)+2|z||w|+(c^2+d^2)) - |(a+c) + (b+d)i|^2\)

\(y = ((a^2+b^2)+2|z||w|+(c^2+d^2)) - ((a+c)^2 + (b+d)^2)\)

\(y = ((a^2+c^2)+2|z||w|+(b^2+d^2)) - ((a+c)^2 + (b+d)^2)\)

\(y = ((a^2+c^2)+2|z||w|+(b^2+d^2)) - ((a^2+2ac+c^2 + (b^2+2bd+d^2))\)

\(y = (2|z||w|) - (2ac +2bd)\)

\(y = (2|z||w|) - (2Re(zw))\)

We've proved this previous in part (b) that \(y\) will always be greater than or equal to 0. Therefore \(|z + w|^2 \leq (|z|+|w|)^2\) is true.
