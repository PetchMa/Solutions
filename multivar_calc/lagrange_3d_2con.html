Lagrange Multipliers - Three Dimensions, Two Constraints

Intro: What are Lagrange multipliers and what's its use?

Check this link for introduction to lagrange multipliers

The lagrange multiplier technique for \(\mathbb{R}^3\) where \(f(x,y,z)\) given two constraint equations requires you to solve a system of 3 equations and 2 variables twice. \(\lambda\) and \(\mu\)

\(\frac{\partial f}{\partial x}= \lambda\frac{\partial g}{\partial x} + \mu\frac{\partial h}{\partial x}\)

\(\frac{\partial f}{\partial y}= \lambda\frac{\partial g}{\partial y} + \mu\frac{\partial h}{\partial y}\)

\(\frac{\partial f}{\partial z}= \lambda\frac{\partial g}{\partial z} + \mu\frac{\partial h}{\partial z}\)

Problem: Use Lagrange multipliers to find the extrema of the function, subject to the given constraint.


\[ f(x,y,z)= x^2+y^2+z^2\] \[ x+y+z=1\] \[ x+2y+3z=6\]

We first need to find the constraint function. Given the constraint \( x+y+z=1\) and \(x+2y+3z=1\) we can arrive at: \[g(x,y,z)=x+y+z-1\] \[h(x,y,z)= x+2y+3z-6\]

The respective partial derivatives of \(f\) and \(g\) are:

\(\frac{\partial f}{\partial x}= 2x\), \(\frac{\partial f}{\partial y}= 2y\), \(\frac{\partial f}{\partial z}= 2z\)

\(\frac{\partial g}{\partial x}=1\), \(\frac{\partial g}{\partial y}=1\), \(\frac{\partial g}{\partial z}=1\)

\(\frac{\partial h}{\partial x}=1\), \(\frac{\partial h}{\partial y}=2\), \(\frac{\partial h}{\partial z}=3\)

Then we set up the system of equations like so

\(\frac{\partial f}{\partial x}= \lambda\frac{\partial g}{\partial x} + \mu\frac{\partial h}{\partial x}\)

\(\frac{\partial f}{\partial y}= \lambda\frac{\partial g}{\partial y} + \mu\frac{\partial h}{\partial y}\)

\(\frac{\partial f}{\partial z}= \lambda\frac{\partial g}{\partial z} + \mu\frac{\partial h}{\partial z}\)

We sub in the values for the partial derivatives.

\(2x= \lambda + \mu \)

\(2y= \lambda +2\mu\)

\(2z= \lambda +3\mu\)

Then we solve for x, y, and z respectively in terms of \(\lambda\) and \(\mu\)

\[x = \frac{1}{2}\lambda + \frac{1}{2}\mu\] \[y = \frac{1}{2}\lambda + \mu\] \[z = \frac{1}{2}\lambda + \frac{3}{2}\mu\]

Then we sub those value into the constraints of \(g(x,y,z)\) and \(h(x,y,z)\)

\[ x+y+z=1\] \[ \frac{1}{2}\lambda + \frac{1}{2}\mu + \frac{1}{2}\lambda + \mu +\frac{1}{2}\lambda + \frac{3}{2}\mu = 1\]

Second equation subsitution

\[ x+2y+3z=6\] \[ \frac{1}{2}\lambda + \frac{1}{2}\mu + \lambda + 2\mu +\frac{3}{2}\lambda + \frac{9}{2}\mu = 1\]

Simplification we get:

\[\frac{3}{2}\lambda + 3\mu = 1\] \[3\lambda + 7\mu = 6\]

Solving that we get\(\lambda = \frac{-22}{3}, \mu = 4\), now we can calculate the values of x, y and z.

\[x = \frac{-5}{3}\] \[y = \frac{1}{3}\] \[z = \frac{7}{3}\]

We can plug the value into the original function and we get \(\frac{25}{3}\) and since we know that the standarded equation of \(f(x,y,z)=x^2+y^2+z^2\) is a standard equation for a parabaloid we can say that \((\frac{-5}{3},\frac{1}{3},\frac{7}{3})\) is a minimum
