Lagrange Multipliers - Three Dimensions, One Constraint

Intro: What are Lagrange multipliers and what's its use?

Check this link for introduction to lagrange multipliers

The lagrange multiplier for \(\mathbb{R}^3\) where \(f(x,y,z)\) given one constraint equation follows the same calculation method as for 2D lagrange multipliers.

\(\frac{\partial f}{\partial x}= \lambda\frac{\partial g}{\partial x}\)

\(\frac{\partial f}{\partial y}= \lambda\frac{\partial g}{\partial y}\)

\(\frac{\partial f}{\partial z}= \lambda\frac{\partial g}{\partial z}\)

Problem: Use Lagrange multipliers to find the extrema of the function, subject to the given constraint.


\[ f(x,y,z)=xyz\] \[ x^2+y^2+z^2=3\]

We first need to find the constraint function. Given the constraint \(x^2+y^2+z^2=3\) we can arrive at: \[g(x,y,z)= x^2+y^2+z^2-3\]

The respective partial derivatives of \(f\) and \(g\) are:

\(\frac{\partial f}{\partial x}= yz\)

\(\frac{\partial f}{\partial y}= xz\)

\(\frac{\partial f}{\partial z}= xy\)

\(\frac{\partial g}{\partial x}=2x\)

\(\frac{\partial g}{\partial y}=2y\)

\(\frac{\partial g}{\partial z}=2z\)

Then we set up the system of equations like so

\(\frac{\partial f}{\partial x}= \lambda\frac{\partial g}{\partial x}\)

\(\frac{\partial f}{\partial y}= \lambda\frac{\partial g}{\partial y}\)

\(\frac{\partial f}{\partial z}= \lambda\frac{\partial g}{\partial z}\)

We sub in the values for the partial derivatives.

\(yz= 2\lambda x \)

\(xz= 2\lambda y\)

\(xy= 2\lambda z\)

Solving this system we get \(\frac{yz}{2x}=\frac{xz}{2y}=\frac{yx}{2z}\). Which gives us \(x=y=z\) Then we sub this new value of x into the original constraint function

\[x^2+y^2+z^2=3\] \[3x^2 = 3\] \[x= \pm 1\]

Therefore our critical values are the following points \((1,1,1)\) and \((-1,-1,-1)\)

Then we check the type of critical values of each point by simply subing in the points into the original function.

\[f(1,1,1) =1 \] \[f(-1,-1,-1) =-1 \]

Therefore, local min at (-1,-1,-1) and local max (1,1,1)
