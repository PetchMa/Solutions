Integrals In Cylindrical Coordinates

Intro: Think of it as 3D polar coordinates. A refresh on polar coordinates here. Cylindrical coordinates can be represented in the following \((r,\theta,z)\)

Conversion between cartesian coordinates and cylindrical coordinates are:

\[x = r\cos(\theta)\] \[y = r\sin(\theta)\] \[z = z\] \[x^2+y^2 =r^2\]

Problem: Convert the following integral into cylindrical coordinates


\[\int_{1}^{3} \int_{-\sqrt{1-y^2}}^{\sqrt{1-y^2}} \int_{\sqrt{x^2+y^2}}^{3}xyz dzdydx \]

We can rewrite the Z bounded integral into the following the \(x^2+y^2 =r^2\) where we can sub the lower bound as r. We can leave the upper bounds as its not in terms of any variables.

Then we need the bounds for the radius. Since we know that for the bounds of x it corresponds to \(\sqrt{1-y^2}\) we can try and solve for radius.

\[x = \sqrt{1-y^2}\] \[r^2\cos^2(\theta) = 1-r^2\sin^2(\theta)\] \[r^2\cos^2(\theta) + r^2\sin^2(\theta) = 1\] \[r^2 = 1\] \[r = \pm 1\]

Finally since we know that the radius is between 0 and 1 since if \[\theta \in [0, 2\pi]\] it will eventually include r=-1.

Therefore the new bounds are: Remember: to include \(r dr\) as for when multiplying for the area you need to account for the extra width.

\[ \int_{2\pi}^{0} \int_{0}^{1} \int_{r}^{3} r\sin(\theta)r\cos(\theta)z r dzdrd\theta \] \[ \int_{2\pi}^{0} \int_{0}^{1} \int_{r}^{3} r^3z\sin(\theta)\cos(\theta) dzdrd\theta \]

Integrals in Spherical Coordinates

Intro: Think of it as coordinates within a globe. it is similar to cylindrical coordinates but a point is represented as \((\rho,\theta,\phi)\)

\[x = \rho\sin(\theta)\cos(\phi)\] \[y = \rho\sin(\theta)\sin(\phi))\] \[z = \rho\cos(\theta)\] \[x^2+y^2+z^2=\rho^2\]

Problem: Convert the following integral into spherical coordinates

\[\int_{1}^{0} \int_{0}^{\sqrt{1-x^2}} \int_{\sqrt{x^2+y^2}}^{\sqrt{1-x^2-y^2}}2 dzdydx \]

We can rewrite the upper and the lower bounds in terms of spherical coordinates.

\[z = \sqrt{1-x^2-y^2}\] \[z^2 = 1-x^2-y^2\] \[(\rho\cos(\theta))^2 = 1-(\rho\sin(\theta)\cos(\phi))^2-(\rho\sin(\theta)\sin(\phi)))^2\] \[\rho^2(\cos^2(\theta)+\sin^2(\theta)\cos^2(\phi)+\sin^2(\theta)\sin^2(\phi)) = 1\] \[\rho^2(\cos^2(\theta)+\sin^2(\theta)(\cos^2(\phi)+\sin^2(\theta))) = 1\] \[\rho^2(\cos^2(\theta)+\sin^2(\theta)(1)) = 1\] \[\rho^2(1) = 1\] \[\rho = \pm 1\]

Given the bounds of \(\rho\) we can then find the bounds for \(\theta\) and \(\phi\). Since both x and y are greater than 0, they are in the first quadrant

\[x,y > 0 \] \[\theta \in [0, \pi/2] \]

To look for the bound of \(\phi\) we need to look at the lower bound of z

\[z = \sqrt{x^2+y^2}\] \[z = \sqrt{r^2}\] \[z = r\]

We know that this exists.

\[z = \rho\cos(\phi)\] \[r = \rho\sin(\phi)\] \[\rho\cos(\phi) = \rho\sin(\phi)\] \[\phi= \pi/4\]

Thus we can rewrite the entire integral as the following: Remember to replace with \(\rho\sin(\phi)\)

\[\int_{1}^{0} \int_{0}^{\sqrt{1-x^2}} \int_{\sqrt{x^2+y^2}}^{\sqrt{1-x^2-y^2}}2 dzdydx = \int_{0}^{\frac{\pi}{4}} \int_\frac{\pi}{2}^{0} \int_{0}^{1}2 \rho \sin(\phi) d\rho d\theta d\phi \]
