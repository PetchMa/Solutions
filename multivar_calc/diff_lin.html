Differential Equations - First Order Linear Equations

Intro: This is a technique for solving first order differential equations when they appear in the product rule form!! This is perhaps one of my favourite ways to solve a first order diff eq as it relies on being tricky with some techniques. This works given the form...

\[\frac{dy}{dx}+P(x)y= Q(x)\]

We can reconize this as the product rule!!! When we try to solve for y. Reminder that this is the product rule.

\[\frac{d}{dx}(f(x)g(x)) = f'(x)g(x) + f(x)g'(x))\]

To get the function above into the form of the product rule all we need to do is multiply by \(e^{\int p(x) dx}\)

\[e^{\int p(x) dx}\frac{dy}{dx}+e^{\int p(x) dx}P(x)y= e^{\int p(x) dx}Q(x)\]

This is when we realize these two functions of product rule matches with the manipulated function.

\[f'(x)g(x) = e^{\int p(x) dx}p(x)y \] \[f(x)g'(x) = e^{\int p(x) dx}\frac{dy}{dx}\] \[ g(x) = y\] \[ g'(x) = \frac{dy}{dx}\] \[f(x) =e^{\int p(x) dx} \] \[f '(x) =e^{\int p(x) dx} \frac{d (\int p(x) dx)}{dx} = e^{\int p(x) dx}p(x) \] \[\frac{d}{dx}(f(x)g(x)) = e^{\int p(x) dx}Q(x)\]

We also know this is true. We can sub in the values of f and g

\[\frac{d}{dx}(e^{\int p(x) dx}y)) = e^{\int p(x) dx}Q(x)\] \[\int \frac{d}{dx}(e^{\int p(x) dx}y)) dx = \int e^{\int p(x) dx}Q(x) dx\] \[e^{\int p(x) dx}y = \int e^{\int p(x) dx}Q(x) dx\] \[y = \frac{\int e^{\int p(x) dx}Q(x) dx}{e^{\int p(x) dx}}\]

Problem: Solve the following differential equation.

\[\frac{dy}{dx} + 3y = 2xe^{-3x}\]

The p value we can identify as the function multiplying y.

\[e^{\int 3 dx}\frac{dy}{dx}+3e^{\int 3 dx}y= e^{\int 3 dx}Q(x)\] \[e^{3x}\frac{dy}{dx}+3e^{3x}y= e^{3x}2xe^{-3x}\] \[ g(x) = y\] \[ g'(x) = \frac{dy}{dx}\] \[f(x) =e^{3x} \] \[f '(x) = 3e^{3x} \] \[\frac{d}{dx}(e^{3x}y) = e^{3x}2xe^{-3x}\] \[\frac{d}{dx}(e^{3x}y) = 2x\] \[\int \frac{d}{dx}(e^{3x}y) dx= \int 2x dx\] \[e^{3x}y= x^2\] \[y= \frac{x^2+c}{e^{3x}}\]

This is the general solution for this differential equation.
