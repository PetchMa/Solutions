Lagrange Multipliers - Two Dimensions, One Constraint

Intro: What are Lagrange multipliers and what's its use?

The lagrange multiplier is a technique of optimization in multivariate calculus given a constraint equation. In regular calculus, we were often given bounds where we'd have to check end points to determine local or global extrema values. However with multivariate functions, we'd often have to check for extrema values of the bounding lines, instead of simply checking end points, requiring more computation. To make it easier we have Lagrange multipliers!

Given the multivariable function \(z = f(x,y)\) and the constraint equation of \(g(x,y)=c\). Take the partial derivatives of each function and set the partial derivatives of \(f\) to the partial derivatives of \(g\) multiplied by \(\lambda\)

\(\frac{\partial f}{\partial x}= \lambda\frac{\partial g}{\partial x}\)

\(\frac{\partial f}{\partial y}= \lambda\frac{\partial g}{\partial y}\)

Given this system of equations we can solve for \(\lambda\) in both equations and set the two equations equal to one another.

Isolate for \(x\) or for \(y\) and subsitue back into the constraint equation and solve for the two variables.

Given the solution, we then check for the type of extrema value using the second partials test

\[D(x,y,\lambda) = \frac{\partial^2 f}{\partial x^2} \frac{\partial^2 f}{\partial y^2} - (\frac{\partial^2 f}{\partial x \partial y})^2 \]

Reminder: The results of the test indicates the following if conditions are met:

  • \(D(x,y) < 0 \) Saddle point, as the concavity of x and y component is complimentry which forms neither a min or a max.
  • \(D(x,y) = 0 \) inconclusive.
  • \(D(x,y) > 0 \) Possible extrema
    • if \(\frac{\partial^2 f}{\partial x^2}\) > 0 we have a min
    • if \(\frac{\partial^2 f}{\partial x^2}\) < 0 we have a max

Problem: Use Lagrange multipliers to find the extrema of the function, subject to the given constraint.


\[ f(x,y)=x+y-5\] \[ xy=1\]

We first need to find the constraint function. Given the constraint \(xy=1\) we can arrive at: \[g(x,y)=xy-1\]

The respective partial derivatives of \(f\) and \(g\) are:

\(\frac{\partial f}{\partial x}= 1\)

\(\frac{\partial f}{\partial y}= 1\)

\(\frac{\partial g}{\partial x}=y\)

\(\frac{\partial g}{\partial y}=x\)

Then we set up the system of equations like so

\(\frac{\partial f}{\partial x}= \lambda\frac{\partial g}{\partial x}\)

\(\frac{\partial f}{\partial y}= \lambda\frac{\partial g}{\partial y}\)

We sub in the values for the partial derivatives.

\(1= \lambda y\)

\(1= \lambda x\)

Solving this system we get \(x=y\). Then we sub this new value of y into the original constraint function

\[x(y) = 1\] \[x^2 = 1\] \[x= \pm 1\]

Therefore our critical values are the following points \((1,1,f(x,y))\) and \((-1,-1,f(x,y))\)

Then we check the type of critical values of each point with the second partials test.

Note: the values return inconclusive when we take the second derivative. Therefore we need to use a different test by forming a bordered Hessian matrix.

\[ H(g,L) = det\begin{bmatrix} 0 & -g_x & -g_y\\[1ex] -g_x & L_{xx} & L_{xy} \\[1ex] -g_y& L_{yx} & L_{yy} \end{bmatrix} \]

Where \(L(x,y,\lambda)\) is defined in terms of when \(\lambda = 1 \) from the first solution \[L(x,y,\lambda) = f(x,y)-\lambda(x,y)\] \[L(x,y,\lambda) = x+y-5-(1)(xy-1)\] \[L(x,y,\lambda) = x+y-xy-4\]

We get the following partial derivatives: \(L_{xx}=0\), \(L_{yy}=0\), \(L_{xy}=L_{yx}=-1\)

\[ H(g,L) = det\begin{bmatrix} 0 & -1 & -1\\[1ex] -1 & 0 & -1 \\[1ex] -1& -1 & 0 \end{bmatrix} = -2 \]

Since \(H(g,L)<0\) (1,1) is a local minimum

Then we find \(L(x,y,\lambda)\) when\(\lambda = -1 \) from the second solution \[L(x,y,\lambda) = f(x,y)-\lambda(x,y)\] \[L(x,y,\lambda) = x+y-5+(1)(xy+1)\] \[L(x,y,\lambda) = x+y+xy-6\]

We get the following partial derivatives: \(L_{xx}=0\), \(L_{yy}=0\), \(L_{xy}=L_{yx}=1\)

\[ H(g,L) = det\begin{bmatrix} 0 & 1 & 1\\[1ex] 1 & 0 & 1 \\[1ex] 1& 1 & 0 \end{bmatrix} = 2 \]

Since \(H(g,L)>0\) (-1,-1) is a local maximum
