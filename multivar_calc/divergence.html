Divergence

Intro: Divergence is similar to curl, except it calculates how much something is diverging from a given point. If a point has positive divergence it would appear as fluid moving away. Negative divergence means fluid is coming together like a hole! Divergence is simply finding the gradient via dot product. This is the flux density of a given region.

\[\vec{F} = \langle F_1, F_2, F_3 \rangle\] \[div \vec{F} = \nabla \cdot \vec{F} \] \[div \vec{F} = \frac{\partial F_1}{\partial x}i + \frac{\partial F_2}{\partial y}j + \frac{\partial F_2}{\partial z}k\]

Probelm: Find the divergence of the following vector field.

\[\vec{F(x,y,z)} = \langle e^x\sin (y), e^y\sin (z), e^z\sin (x) \rangle\] \[div \vec{F} = \nabla \cdot \vec{F} \] \[div \vec{F} = \frac{\partial F_1}{\partial x}i + \frac{\partial F_2}{\partial y}j + \frac{\partial F_2}{\partial z}k\] \[div \vec{F} = (e^x\sin (y))i + (e^y\sin (z))j + (e^z\sin (x))k\]
