Differential Equations - Second Order Homogenous

Intro: With second order differential equations you have functions with second order derivatives which makes it difficult to solve with first order techniques. Thus we need to solve these differential equations in other ways. Given a homogenous equation , an equation that equals 0, we can solve these with this given formula.

Let \(r\) be the roots of a second order homogenous equation. We replace y with \(r\) raised the to same power as the order of differentiation.

\[ay'' + by'+cy = 0\]

We would replace y with the following

\[ar^2+ br+c = 0\] \[r = \frac{-b\pm\sqrt{b^2-4ac}}{2a}\]

With the given r values you can sub it into this equation:...

\[y(x) = c_1y^{r_1x}+c_2e^{r_2x}\]

This is only if the equation gives us real and distinct roots. If you get equal roots, we take the following equation =>

\[y(x) = c_1y^{rx}+c_2xe^{rx}\]

If we have a complex conjugate root, then we use the following:

\[r = \alpha \pm \beta i\] \[y(x) = e^{\alpha x}[c_1\cos(\beta x)+ c_2 \sin(\beta x)]\]

Mini proof for distinct roots: This technique works because => if we set that \(r_{1}\) and \(r_{2}\) are the roots to the equation \(ar^2+br+c=0\).

\[ ay''+by'+cy =0\] \[y = e^{r_1x+c_1} + e^{r_2x+c_2}\] \[y' = r_1e^{r_1x} + r_2e^{r_2x}\] \[y'' = r_1^2e^{r_1x} + r_2^2e^{r_2x}\]

Now if we sub into the orignal equation

\[ LS=a( r_1^2e^{r_1x} + r_2^2e^{r_2x})+ b(r_1e^{r_1x} + r_2e^{r_2x})+c(e^{r_1x+c_1} + e^{r_2x+c_2})\] \[ =ar_1^2e^{r_1x} + ar_2^2e^{r_2x}+ br_1e^{r_1x} + br_2e^{r_2x}+ce^{r_1x+c_1} + ce^{r_2x+c_2} \]

We can rearrange the equation to the following.

\[ =[ar_1^2e^{r_1x} +br_1e^{r_1x} +ce^{r_1x+c_1} ]+ [ar_2^2e^{r_2x}+ br_2e^{r_2x}+ ce^{r_2x+c_2} ]\]

Since from each trinomial group we can take out common factors.

\[= e^{r_1x}[ar_1^2 +br_1 +ce^{c_1} ]+ e^{r_2x}[ar_2^2+ br_2+ ce^{c_2} ]\]

Since we don't know what the values of the constants are we can set them to zero, but they will be taken into account when we solve the differential equation.

\[=e^{r_1x}[ar_1^2 +br_1 +c]+ e^{r_2x}[ar_2^2+ br_2+ c ]\] \[=e^{r_1x}[0]+ e^{r_2x}[0 ]\]

We zero out the entire equation.

\[=0\] \[LS=RS\]
