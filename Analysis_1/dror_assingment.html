1. Show that if \(a>0\) then \(ax^2+bx+c \geq 0\) given that \(b^2-4ac \leq 0\).

Let \(p(x) = ax^2+bx+c \)

We can rewrite the function as the following... by completeling the square.

\[p(x) = a(x-\frac{b}{2a})^2+(c-\frac{b^2}{4a})\]

We know that the term \(a(x-\frac{b}{2a})^2 > 0\) since we know that \(a > 0\) and that the square of any number is positve. Thus the entire term is positive.

We also know the following is true from the question. \(b^2-4ac \leq 0\) which implies \(b^2 \leq 4ac\) which suggests \(\frac{b^2}{2a} \leq c\)

This suggests that the term \((c-\frac{b^2}{4a}) \geq 0\).

Since if we added only positve or 0 terms together, we will always get a positive or 0 sum. Then \(ax^2+bx+c \geq 0\). Thus its proven. QED
