Problem 2

Consider:\(g(x) =x^2\) and the function

\[ \begin{cases} 0, & y \in \mathbb{Q}\\ 1, & y\in \mathbb{Q'}\\ \end{cases} \]

Part A

Find all \(h(y) \leq y\)

We know that \(\forall y \in \mathbb{Q} [0,\infty)\) that \(y\geq 0\). The function \(h(y)\) sends y \(h(y) = 0\)) Thus \(h(y) \leq y\). We also know that \(\forall y \in \mathbb{Q'} [1,\infty)\) that \(y\geq 1\). The function \(h(y)\) sends y \(h(y) = 1\)) Thus \(h(y) \leq y\)

Part B

For which \(y\) does \(h(y) \leq g(y)\)

We know that \(\forall y \in \mathbb{Q} \) that \(g(y) \geq 0\) because its a square. We know that \(\forall y \in \mathbb{Q} , h(y)=0\). Thus \(h(y) \leq g(y)\) In this case.

We know that \(\forall y \in \mathbb{Q'}(-\infty, -1] \cup [1, \infty) \) that \(g(y) \geq 1\) because its a square. We know that \(\forall y \in \mathbb{Q'} , h(y)=1\). Thus \(h(y) \leq g(y)\) In this case.

Part C

What is \(g(h(z)) - h(z)\)?

Since \(h(z)\) can only take on either \(1\) or \(0\). \(g(x)\) would thus only take on either \(1\) or \(0\). Thus \(g(h(z)) - h(z) = 0\). No matter what number \(z\) takes on, the square of \(1,0\) is always the same.

Part D

For which \(w\) is \(g(w) \leq w\)?

We know that the product of numbers greater than 1 will always be greater than the factors of the product. Thus if \(|w|>1\) then \(w^2> w\). Not represented in the inequality.

We know that if each factor is \(\leq 1\) then the product is less than its factors. \(|w|\leq 1 \Rightarrow w^2 \leq w\). Thus when \(w\in \mathbb{R}[-1,1]\) the inequality holds true.

Part E

For which \(\epsilon\) is \(g(g(\epsilon)) = g(\epsilon)\)?

We know that the expression says \((x^2)^2 = x^2 \) which implies \(x^2 =1\). We know then that the only 2 values \(\epsilon\) can take on is \(\epsilon = \pm 1\)


Problem 16 a)

Suppose \(f\) satisfies \(f(x+y) = f(x)+f(y)\) for all \(x\) and \(y\). Prove that \(f(x_1+x_2+...x_n) = f(x_1)+f(x_2)+...\)

\[f(\sum_{k=1}^{n+1} x_k) =\sum_{k=1}^{n+1} f(x_k) \]

Prove by Induction

Base case

We are told that the base case is true. It is stated that \(f\) satisfies \(f(x+y) = f(x)+f(y)\) which implies \(f\) satisfies \(f(x_1+x_2) = f(x_1)+f(x_2)\)

General case

We assume that \(f\) satisfies \(f(x_1+x_2+...x_n) = f(x_1)+f(x_2)+...\)

\[f(\sum_{k=1}^{n+1} x_k) =\sum_{k=1}^{n+1} f(x_k) \]

Induction case

We can test to see if the \(n+1\) case holds up. Let \(y = x_1+x_2+...x_{n}\)

\[f(x_1+x_2+...x_{n+1}) =f(y+x_{n+1})\]

We know from the base case that this is true.

\[f(x_1+x_2+...x_{n+1}) =f(y) + f(x_{n+1})\]

Sub back the value for \(y\)

\[f(x_1+x_2+...x_{n+1}) =f(x_1+x_2+...x_{n}) + f(x_{n+1})\]

We already assumed that \(f(x_1+x_2+...x_n) = f(x_1)+f(x_2)+...\)

\[f(x_1+x_2+...x_{n+1}) = f(x_1)+f(x_2)+...+ f(x_{n+1})\]

By induction we've therefore proven that this is true.

Problem 19

Prove that there exists no function \(f,g\) such that...

\[f(x)+g(y) = xy\]

Or...

\[f(x)g(y) = x+y\]

First relationship

We first prove the first relation does not allow \(f,g\) to be a function.

We know that the following is true.

\[f(x)+g(0)=0\quad and \quad f(0)+g(y)=0\]

Thus if we add them together, they must equal to zero.

\[[f(x)+g(0)] + [f(0)+g(y)]=0\] \[[f(x)+g(y)] +[f(0)+g(0)] =0\] \[[f(x)+g(y)] +0 =0\] \[f(x)+g(y)=0\]

This shows that

\[f(x)+g(y) = xy \quad and \quad f(x)+g(y)=0, \quad \forall x,y\in\mathbb{R}\]

This is the issue. If we hold the function \(g\) constant such that \(g(\alpha)\).

\[f(x)+g(\alpha)=x+\alpha\]

But we've also shown that \(f(x)+g(y)=0, \quad \forall x,y\in\mathbb{R}\)

\[f(x)+g(\alpha)=0\]

Since \(x\) can take on any value, \(x+\alpha \neq 0\). Thus \(f\) is not a function. Since the function can equal any real number and 0 at the SAME time.

If we hold the function\(f\) constant such that \(f(\alpha)\).

\[f(\alpha)+g(x)=\alpha+x\]

But we've also shown that \(f(x)+g(y)=0, \quad \forall x,y\in\mathbb{R}\)

\[f(\alpha)+g(x)=0\]

Since \(x\) can take on any value, \(x+\alpha \neq 0\). Thus \(g\) is also not a function. Since the function can equal any real number and 0 at the SAME time.

Second relationship

We know that the following is true.

\[f(-x)g(x) =0\] \[f(-y)g(y) =0\]

What if we multiplied them together?

\[f(-y)g(y) f(-x)g(x)=0 \] \[f(-y) f(-x)g(y)g(x)=0 \] \[(-x-y)(x+y)=0 \] \[-(x+y)^2=0 \] \[-(f(x)g(y))^2=0 \] \[f(x)g(y)=0 \]

This is the problem. We've previously defined that \(f(x)g(y) = x+y\)

If we hold one of these constant.

\[f(x)g(\alpha) = x+\alpha\]

We also know that its equal to 0 at the same time from our previous proof. \(f\) is not a function. Since the function can equal any real number and 0 at the SAME time.

\[f(\alpha)g(x) = \alpha+x\]

We also know that its equal to 0 at the same time from our previous proof. \(g\) is not a function. Since the function can equal any real number and 0 at the SAME time.

Therefore we've proven that neither of these definitions allow \(f,g\) to be a function.

Problem 26

Suppose \(f(g(x))=I\) and that \(h(f(x)) = I\) prove that \(h=g\)

We know the following is true. We use the identity property. [thanks to Mojan :)]

\[(h \circ f) \circ g = h \circ (f \circ g)\] \[I \circ g = h \circ I\]

We replaced it with the indentity function. We can then conclude that these two are the same.

\[ g = h \]

Second way of proving.

\[h^{-1}\circ f\circ g\circ g^{-1}\circ f^{-1}\circ h (x) = x\]

This is based on the associative property of composition.

We can also rewrite \(f(g(x)) = h(f(x))\) which we can then write as \(h^{-1}(f(g(x))) = f(x) \). Thus we can replace

\[f\circ g^{-1}\circ f^{-1}\circ h (x) = x\]

From this we can say that the inner composition is the inverse of the outer.

\[f \circ g^{-1}= (f^{-1}\circ h (x))^{-1}\]

We can say that the inverse is distributive because of this proof.

\[f^{-1}\circ g^{-1}(g\circ f(x))=x\]

We can also say that the following is true based on the grouping

\[(f\circ g)^{-1}(g\circ f(x))=x\]

Thus the following is true.

\[(f\circ g)^{-1}(g\circ f(x))=f^{-1}\circ g^{-1}\circ (g\circ f(x))\] \[(f\circ g)^{-1}f^{-1} \circ g^{-1}\]

Thus if we go back and distribute it in. We get...

\[f\circ g^{-1}= f\circ h (x)^{-1}\]

Take the inverse of \(f\) on both sides.

\[g^{-1}= h (x)^{-1} \Rightarrow g=h \]

Proven.
