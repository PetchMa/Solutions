Problem 1.ii

Prove: that \(1^3+2^3+4^3+...n^3 = (1+2+3+4+5+..+n)^2\)

\[\sum_{k=1}^{n}k^3 = (\sum_{k=0}^{n+1}k)^2\]

Prove by Induction.

Base Case

We prove the base case. If \(\sum_{k=1}^{n}k^3 = (\sum_{k=0}^{n+1}k)^2\), then \( \sum_{k=1}^{n}k^3 -(\sum_{k=0}^{n+1}k)^2 = 0\) for \(n=1\)

\[\sum_{k=1}^{n}k^3 - (\sum_{k=0}^{n+1}k)^2 = 1^3-1^2\] \[= 0\]

The base case is true.

Assume that \(S(n) = \sum_{k=1}^{n}k^3 = (\sum_{k=0}^{n}k)^2\) is true for \(n\).

Induction Step

Prove the \(n+1\) case. Let \(S(n) =\sum_{k=1}^{n}k^3 \) and \(S'(n) = (\sum_{k=0}^{n+1}k)^2\). If \(S(n+1) = S'(n+1) \Rightarrow S(n+1)-S'(n+1) = 0\)

We know that \(S(n+1) = S(n)+(n+1)^3\)

\[S(n+1)-S'(n+1) = S(n)+(n+1)^3 -S'(n+1) \]

We assumed that \(S(n) = (\sum_{k=0}^{n+1}k)^2\)

\[S(n+1)-S'(n+1) = (\sum_{k=0}^{n}k)^2+(n+1)^3 - (\sum_{k=0}^{n+1}k)^2 \]

Notice that the summation is basically a sum of natural numbers all squared. Since we proved that the sum of \(n\) natural numbers is \(\frac{n(n+1)}{2}\) we can sub for that. Same for the \(n+1\) case.

\[S(n+1)-S'(n+1) = (\frac{n(n+1)}{2})^2+(n+1)^3 - (\frac{(n+1)(n+2)}{2})^2 \] \[S(n+1)-S'(n+1) = (\frac{n^2+n}{2})^2+(n+1)^3 - (\frac{n^2+3n+2}{2})^2 \] \[S(n+1)-S'(n+1) = \frac{n^4+2n^3+n^2}{4}+(n+1)^3 - \frac{n^4+6n^3+13n^2+12n+4}{4} \]

Subtract the polynomials.

\[S(n+1)-S'(n+1) = \frac{-4n^3-12n^2-12n-4}{4} +(n+1)^3\] \[S(n+1)-S'(n+1) = -n^3-3n^2-3n-1 +(n+1)^3\]

Factor.

\[S(n+1)-S'(n+1) = -(n+1)^3 +(n+1)^3\] \[S(n+1)-S'(n+1) = 0\] \[\therefore S(n+1) = S'(n+1) \]

Thus we've proven via induction that \(1^3+2^3+4^3+...n^3 = (1+2+3+4+5+..+n)^2\) is true.

Problem 2.ii

Derive: the formula for the following series.

\[\sum_{i=1}^n (2i-1)^2 = 1^2+3^2+5^2 + .. (2n-1)^2\]

We can expand the inner bracket

\[\sum_{i=1}^n (2i-1)^2 =\sum_{i=1}^n 4i^2-4i+1 \]

We can break up the summation.

\[\sum_{i=1}^n (2i-1)^2 =\sum_{i=1}^n 4i^2-\sum_{i=1}^n4i+\sum_{i=1}^n 1 \]

Move coefficents out of the summations

\[\sum_{i=1}^n (2i-1)^2 =4\sum_{i=1}^n i^2-4\sum_{i=1}^n i+\sum_{i=1}^n 1 \]

Notice that these are just the sum of squares, the sum of natural numbers and the last is just adding 1 \(n\) times which is \( = n\)

We've previously proved the sum of squares and the sum of natural numbers so we just plug in.

\[\sum_{i=1}^n (2i-1)^2 =4(\frac{n(n+1)(2n+1)}{6})-4(\frac{n(n+1)}{2}) +n\] \[\sum_{i=1}^n (2i-1)^2 =2(\frac{2 n^3 + 3 n^2 + n}{3})-(2n^2+n)\] \[\sum_{i=1}^n (2i-1)^2 =\frac{4 n^3 + 6 n^2 + 2n-6n^2-3n}{3}\] \[\sum_{i=1}^n (2i-1)^2 =\frac{4 n^3 -n}{3}\] \[\sum_{i=1}^n (2i-1)^2 =\frac{n(2 n-1)(2n+1)}{3}\]

This is the formula.

Problem 3 d)

Prove: the binomial theorem which states the following:

\[(x+y)^n = \sum_{k=0}^n {n\choose k}x^ky^{n-k}\]

Proof by Induction.

Base case

We prove the base case. If \(\sum_{k=0}^n {n\choose k}x^ky^{n-k} = (x+y)^n, \quad n=1\), then \( \sum_{k=0}^n {n\choose k}x^ky^{n-k} - (x+y)^n = 0\) for \(n=1\)

\[\sum_{k=0}^n {n\choose k}x^ky^{n-k} - (x+y)^n ={1\choose 0}x + {1\choose 1} y - (x+y) = 0\]

Therefore we proven the base case.

Assume that \(\sum_{k=0}^n {n\choose k}x^ky^{n-k} = (x+y)^n\) is true for \(n\).

Induction Step

If the \(n+1\) step is true then, \((x+y)^{n+1} - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k} = 0 \)

We rewrite the statement.

\[(x+y)^{n+1} - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k} =(x+y)^{n}(x+y) - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k+1} \]

Sub in our assumption.

\[ =(\sum_{k=0}^n {n\choose k}x^ky^{n-k})(x+y) - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k+1} \]

Distribute

\[ =(\sum_{k=0}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=0}^n {n\choose k}x^ky^{n-k+1}) - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k+1} \]

We know the identity \({n+1\choose k} = {n\choose k-1}+ {n\choose k}\). We sub it in.

\[ =(\sum_{k=0}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=0}^n {n\choose k}x^ky^{n-k+1}) - (\sum_{k=0}^{n+1} {n\choose k-1}x^ky^{n-k+1}+ {n\choose k}x^ky^{n-k+1}) \]

NOTE: there is something wrong with our sum... we can't take \(k< 0\)! Nor can we take the \({n\choose n+1}\)... so we are going to deal with those two cases manually add them in.

\[ =(\sum_{k=0}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=0}^n {n\choose k}x^ky^{n-k+1}) - [\sum_{k=1}^{n} {n+1\choose k}x^ky^{n-k+1} + {n+1\choose 0}x^0y^{n+1} + {n+1\choose n+1}x^{n+1}]\] \[ =(\sum_{k=0}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=0}^n {n\choose k}x^ky^{n-k+1}) - [\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1}+ \sum_{k=1}^{n}{n\choose k}x^ky^{n-k+1}+ y^{n+1} + x^{n+1}]\]

Lets manipulate the first part to have the same limits.

\[ =(\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=1}^n {n\choose k}x^ky^{n-k+1})+{n\choose 0}x^{0+1}y^{n-0}+{n\choose 0}x^0y^{n-0+1} - [(\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1}+ \sum_{k=1}^{n}{n\choose k}x^ky^{n-k+1}+ y^{n+1} + x^{n+1}]\] \[ =(\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=1}^n {n\choose k}x^ky^{n-k+1})+xy^{n}+y^{n+1} - [(\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1}+ \sum_{k=1}^{n}{n\choose k}x^ky^{n-k+1}+ y^{n+1} + x^{n+1}]\]

We can subtract from the expression.

\[ =\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k}+xy^{n} - [(\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1} + x^{n+1}]\]

Okay what if we wrote out the first few terms of each summation? What do we get??

\[\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k} = {n\choose 1}x^{2}y^{n-1}+{n\choose 2}x^{3}y^{n-2}+....+{n\choose n-1}x^ny^1 + {n\choose n} x^{n+1}\] \[\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1} = {n\choose 0}x^1y^{n}+{n\choose 1}x^{2}y^{n-1}+{n\choose 2}x^{3}y^{n-2}+...+{n\choose n-1}x^ny^1 \]

Notice how each term cancels out except for \({n\choose 0} xy^{n}\) and \({n\choose n} x^{n+1}\). Notice how those terms can be canceled as well! They appear in the equation above.Thus we get...

\[ =\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k}+xy^{n} - [(\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1} + x^{n+1}] = 0\]

Prove: the following summation.

\[\sum_{k=0}^n {n\choose k}= 2^n\]

We will prove using binomial theorem

We know that the binomial theorem is the following.

\[(x+y)^n = \sum_{k=0}^n {n\choose k}x^ky^{n-k}\]

Notice that we can rewrite the expression is equal to ...

\[\sum_{k=0}^n {n\choose k}= \sum_{k=0}^n {n\choose k} \cdot 1\cdot 1\] \[\sum_{k=0}^n {n\choose k}= \sum_{k=0}^n {n\choose k} \cdot 1^k\cdot 1^{n-k}\]

Its in the form of the binomial theorem and we can rewrite it as the following..

\[\sum_{k=0}^n {n\choose k}= (1+1)^n\] \[\sum_{k=0}^n {n\choose k}= 2^n\]
