Problem 1.ii

Prove: that \(1^3+2^3+4^3+...n^3 = (1+2+3+4+5+..+n)^2\)

\[\sum_{k=1}^{n}k^3 = (\sum_{k=0}^{n+1}k)^2\]

Prove by Induction.

Base Case

We prove the base case. If \(\sum_{k=1}^{n}k^3 = (\sum_{k=0}^{n+1}k)^2\), then \( \sum_{k=1}^{n}k^3 -(\sum_{k=0}^{n+1}k)^2 = 0\) for \(n=1\)

\[\sum_{k=1}^{n}k^3 - (\sum_{k=0}^{n+1}k)^2 = 1^3-1^2\] \[= 0\]

The base case is true.

Assume that \(S(n) = \sum_{k=1}^{n}k^3 = (\sum_{k=0}^{n}k)^2\) is true for \(n\).

Induction Step

Prove the \(n+1\) case. Let \(S(n) =\sum_{k=1}^{n}k^3 \) and \(S'(n) = (\sum_{k=0}^{n+1}k)^2\). If \(S(n+1) = S'(n+1) \Rightarrow S(n+1)-S'(n+1) = 0\)

We know that \(S(n+1) = S(n)+(n+1)^3\)

\[S(n+1)-S'(n+1) = S(n)+(n+1)^3 -S'(n+1) \]

We assumed that \(S(n) = (\sum_{k=0}^{n+1}k)^2\)

\[S(n+1)-S'(n+1) = (\sum_{k=0}^{n}k)^2+(n+1)^3 - (\sum_{k=0}^{n+1}k)^2 \]

Notice that the summation is basically a sum of natural numbers all squared. Since we proved that the sum of \(n\) natural numbers is \(\frac{n(n+1)}{2}\) we can sub for that. Same for the \(n+1\) case.

\[S(n+1)-S'(n+1) = (\frac{n(n+1)}{2})^2+(n+1)^3 - (\frac{(n+1)(n+2)}{2})^2 \] \[S(n+1)-S'(n+1) = (\frac{n^2+n}{2})^2+(n+1)^3 - (\frac{n^2+3n+2}{2})^2 \] \[S(n+1)-S'(n+1) = \frac{n^4+2n^3+n^2}{4}+(n+1)^3 - \frac{n^4+6n^3+13n^2+12n+4}{4} \]

Subtract the polynomials.

\[S(n+1)-S'(n+1) = \frac{-4n^3-12n^2-12n-4}{4} +(n+1)^3\] \[S(n+1)-S'(n+1) = -n^3-3n^2-3n-1 +(n+1)^3\]

Factor.

\[S(n+1)-S'(n+1) = -(n+1)^3 +(n+1)^3\] \[S(n+1)-S'(n+1) = 0\] \[\therefore S(n+1) = S'(n+1) \]

Thus we've proven via induction that \(1^3+2^3+4^3+...n^3 = (1+2+3+4+5+..+n)^2\) is true.

Problem 2.ii

Derive: the formula for the following series.

\[\sum_{i=1}^n (2i-1)^2 = 1^2+3^2+5^2 + .. (2n-1)^2\]

We can expand the inner bracket

\[\sum_{i=1}^n (2i-1)^2 =\sum_{i=1}^n 4i^2-4i+1 \]

We can break up the summation.

\[\sum_{i=1}^n (2i-1)^2 =\sum_{i=1}^n 4i^2-\sum_{i=1}^n4i+\sum_{i=1}^n 1 \]

Move coefficents out of the summations

\[\sum_{i=1}^n (2i-1)^2 =4\sum_{i=1}^n i^2-4\sum_{i=1}^n i+\sum_{i=1}^n 1 \]

Notice that these are just the sum of squares, the sum of natural numbers and the last is just adding 1 \(n\) times which is \( = n\)

We've previously proved the sum of squares and the sum of natural numbers so we just plug in.

\[\sum_{i=1}^n (2i-1)^2 =4(\frac{n(n+1)(2n+1)}{6})-4(\frac{n(n+1)}{2}) +n\] \[\sum_{i=1}^n (2i-1)^2 =2(\frac{2 n^3 + 3 n^2 + n}{3})-(2n^2+n)\] \[\sum_{i=1}^n (2i-1)^2 =\frac{4 n^3 + 6 n^2 + 2n-6n^2-3n}{3}\] \[\sum_{i=1}^n (2i-1)^2 =\frac{4 n^3 -n}{3}\] \[\sum_{i=1}^n (2i-1)^2 =\frac{n(2 n-1)(2n+1)}{3}\]

This is the formula.

Problem 3 d)

Prove: the binomial theorem which states the following:

\[(x+y)^n = \sum_{k=0}^n {n\choose k}x^ky^{n-k}\]

Proof by Induction.

Base case

We prove the base case. If \(\sum_{k=0}^n {n\choose k}x^ky^{n-k} = (x+y)^n, \quad n=1\), then \( \sum_{k=0}^n {n\choose k}x^ky^{n-k} - (x+y)^n = 0\) for \(n=1\)

\[\sum_{k=0}^n {n\choose k}x^ky^{n-k} - (x+y)^n ={1\choose 0}x + {1\choose 1} y - (x+y) = 0\]

Therefore we proven the base case.

Assume that \(\sum_{k=0}^n {n\choose k}x^ky^{n-k} = (x+y)^n\) is true for \(n\).

Induction Step

If the \(n+1\) step is true then, \((x+y)^{n+1} - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k} = 0 \)

We rewrite the statement.

\[(x+y)^{n+1} - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k} =(x+y)^{n}(x+y) - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k+1} \]

Sub in our assumption.

\[ =(\sum_{k=0}^n {n\choose k}x^ky^{n-k})(x+y) - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k+1} \]

Distribute

\[ =(\sum_{k=0}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=0}^n {n\choose k}x^ky^{n-k+1}) - \sum_{k=0}^{n+1} {n+1\choose k}x^ky^{n-k+1} \]

We know the identity \({n+1\choose k} = {n\choose k-1}+ {n\choose k}\). We sub it in.

\[ =(\sum_{k=0}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=0}^n {n\choose k}x^ky^{n-k+1}) - (\sum_{k=0}^{n+1} {n\choose k-1}x^ky^{n-k+1}+ {n\choose k}x^ky^{n-k+1}) \]

NOTE: there is something wrong with our sum... we can't take \(k< 0\)! Nor can we take the \({n\choose n+1}\)... so we are going to deal with those two cases manually add them in.

\[ =(\sum_{k=0}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=0}^n {n\choose k}x^ky^{n-k+1}) - [\sum_{k=1}^{n} {n+1\choose k}x^ky^{n-k+1} + {n+1\choose 0}x^0y^{n+1} + {n+1\choose n+1}x^{n+1}]\] \[ =(\sum_{k=0}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=0}^n {n\choose k}x^ky^{n-k+1}) - [\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1}+ \sum_{k=1}^{n}{n\choose k}x^ky^{n-k+1}+ y^{n+1} + x^{n+1}]\]

Lets manipulate the first part to have the same limits.

\[ =(\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=1}^n {n\choose k}x^ky^{n-k+1})+{n\choose 0}x^{0+1}y^{n-0}+{n\choose 0}x^0y^{n-0+1} - [(\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1}+ \sum_{k=1}^{n}{n\choose k}x^ky^{n-k+1}+ y^{n+1} + x^{n+1}]\] \[ =(\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k})+(\sum_{k=1}^n {n\choose k}x^ky^{n-k+1})+xy^{n}+y^{n+1} - [(\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1}+ \sum_{k=1}^{n}{n\choose k}x^ky^{n-k+1}+ y^{n+1} + x^{n+1}]\]

We can subtract from the expression.

\[ =\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k}+xy^{n} - [(\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1} + x^{n+1}]\]

Okay what if we wrote out the first few terms of each summation? What do we get??

\[\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k} = {n\choose 1}x^{2}y^{n-1}+{n\choose 2}x^{3}y^{n-2}+....+{n\choose n-1}x^ny^1 + {n\choose n} x^{n+1}\] \[\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1} = {n\choose 0}x^1y^{n}+{n\choose 1}x^{2}y^{n-1}+{n\choose 2}x^{3}y^{n-2}+...+{n\choose n-1}x^ny^1 \]

Notice how each term cancels out except for \({n\choose 0} xy^{n}\) and \({n\choose n} x^{n+1}\). Notice how those terms can be canceled as well! They appear in the equation above.Thus we get...

\[ =\sum_{k=1}^n {n\choose k}x^{k+1}y^{n-k}+xy^{n} - [(\sum_{k=1}^{n} {n\choose k-1}x^ky^{n-k+1} + x^{n+1}] = x^{n+1}+xy^{n} - x^{n+1} -xy^{n} \] \[=0\]

Therefore by induction we've proven the binomial theorem is true.

Problem 6.i

Derive: the general solution to the following sum of cubes

\[1^3 + 2^3 + 3^3 +.... + n^3\]

We start off with this function. We know that the following is true. Distribute it out.

\[(k+1)^4 - k^4 = 4k^3+6k^2+4k+1\]

What if we added up every value of \(k \in [1,n]\)? What would we get?

\[\sum_{k=1}^n (k+1)^4-k^4 = (2^4-1^4)+(3^4-2^4)+(4^4-3^4).....+(n^4-(n-1))+((n+1)^4-n^4)\]

Notice that every term cancels except for \((n+1)^4-1^4\). Thus we collapsed the sumation.

\[\sum_{k=1}^n (k+1)^4-k^4 = (n+1)^4-1^4\]

But we also know that \((k+1)^4 - k^4 = 4k^3+6k^2+4k+1\) what if we plug that in?

\[\sum_{k=1}^n 4k^3+6k^2+4k+1 = (n+1)^4-1^4\]

Distribute the summation.

\[\sum_{k=1}^n 4k^3+\sum_{k=1}^n6k^2+ \sum_{k=1}^n4k+ \sum_{k=1}^n1 = (n+1)^4-1^4\] \[4\sum_{k=1}^n k^3+6\sum_{k=1}^nk^2+ 4\sum_{k=1}^nk+ \sum_{k=1}^n 1 = (n+1)^4-1^4\]

Notice that we know what the sum of squares and sum of natural numbers equals to.

\[4\sum_{k=1}^n k^3+6 \frac{n(n+1)(2n+1)}{6}+ 4\frac{n(n+1)}{2}+ n = (n+1)^4-1^4\]

We can rearrange.

\[4\sum_{k=1}^n k^3+n(n+1)(2n+1)+ 2n(n+1)+ n = (n+1)^4-1^4\] \[4\sum_{k=1}^n k^3 = (n+1)^4-1^4 - n -2n(n+1) - n(n+1)(2n+1) \] \[\sum_{k=1}^n k^3 = \frac{(n+1)^4-1^4 - n -2n(n+1) - n(n+1)(2n+1) }{4} \]

We simplify and we get...

\[\sum_{k=1}^n k^3 = \frac{1}{4} n^2 (n^2 + 2 n + 1) = (\frac{n(n+1)}{2})^2\]

Thats the general formula for the sum of cubes.

Problem 7

Show that: the general solution to the following sum of powers to the \(p\) can be written as

\[\sum_{k=1}^n k^p=1^p + 2^p +.... + n^p = \frac{n^{p+1}}{p+1}+An^p + Bn^{p-1}+ ... \]

Proof by Induction.

Base case

We can check the base case when \(p=1\), which we know from the general formula equals...

\[\sum_{k=1}^n k^1=1 + 2 +.... + n = \frac{1}{2}n^2+\frac{1}{2}n\]

This evidently follows the form \(\frac{n^{p+1}}{p+1}+An^p + Bn^{p-1}+ ... =\frac{n^{1+1}}{1+1}+An^p + Bn^{p-1}+ ... = \frac{n^{2}}{2}+An + B\)

Assume General case

Assume that for \(p-1\) and all cases \(< p-1\) also follows the form.

\[\sum_{k=1}^n k^{p-1} =1^{p-1} + 2^{p-1} +.... + n^{p-1} = \frac{n^{p}}{p}+An^{p-1} + Bn^{p-2}+ ... \]

Induction Step

We prove the \(p\)th case.

We use the same method as before in problem 6. We know this is true from the binomial theorem.

\[(k+1)^{p+1} - k^{p+1} = \sum_{i=1}^{p+1} {p+1\choose i}k^{p-i+1}\]

If we added up everything such that ...

\[\sum_{k=1}^n (k+1)^{p+1}-k^{p+1} = (2^{p+1}-1^{p+1})+(3^{p+1}-2^{p+1})+(4^{p+1}-3^{p+1}).....+(n^{p+1}-(n-1))+((n+1)^{p+1}-n^{p+1})\]

Notice that every term cancels except for \((n+1)^{p+1}-1^{p+1}\). Thus we collapsed the sumation.

\[\sum_{k=1}^n (k+1)^{p+1}-k^{p+1} = (n+1)^{p+1}-1\]

We can plug in our equation from before.

\[\sum_{k=1}^n (\sum_{i=1}^{p+1} {p+1\choose i}k^{p-i+1}) = (n+1)^{p+1}-1\]

We can swap the summation notation.

\[\sum_{i=1}^{p+1} (\sum_{k=1}^n {p+1\choose i}k^{p-i+1}) = (n+1)^{p+1}-1\] \[\sum_{i=1}^{p+1}( {p+1\choose i} \sum_{k=1}^n k^{p-i+1})) = (n+1)^{p+1}-1\]

Notice we've broken it down to sum of various powers.

All we care about however is the sum for \(i=1\). Lets bring that out.

\[( {p+1\choose 1} \sum_{k=1}^n k^{p})) +\sum_{i=1}^{p+1}( {p+1\choose i} \sum_{k=1}^n k^{p-i+1})) = (n+1)^{p+1}-1\]

Since we only care about the sum of \(p\) powers. We rearrange for that sum.

\[ {p+1\choose 1} \sum_{k=1}^n k^{p} = (n+1)^{p+1}-1 - \sum_{i=1}^{p+1}( {p+1\choose i} \sum_{k=1}^n k^{p-i+1}))\]

Binomial theorem.

\[ (p+1)\sum_{k=1}^n k^{p} =(\sum_{k=0}^{p+1} {p+1\choose k} n^{p+1-k})-1 - \sum_{i=1}^{p+1}( {p+1\choose i} \sum_{k=1}^n k^{p-i+1}))\]

Notice that the term \(\sum_{k=1}^n k^{p-i+1}\) we've already assumed to be capable of writting in the form \(\frac{n^{p}}{p}+An^{p-1} + Bn^{p-2}+ ...\) We know that the sum of polynomials will always be a polynomial. Thus \( \sum_{i=1}^{p+1}( {p+1\choose i} \sum_{k=1}^n k^{p-i+1}))\) will also be polynomial. This satisfies the requirement that the general solution is polynomial.

We know that \(\sum_{k=0}^{p+1} {p+1\choose k} n^{p+1-k}\) will contain the term \(n^{p+1}\). Which is the first term.

We can rearrange the leading coefficents into the following form. Which shows that the term \( \frac{n^{p+1}}{p+1}\) would thus be included.

\[ \sum_{k=1}^n k^{p} = \frac{(\sum_{k=0}^{p+1} {p+1\choose k} n^{p+1-k})-1 - \sum_{i=1}^{p+1}( {p+1\choose i} \sum_{k=1}^n k^{p-i+1}))}{p+1}\]

Since the form only requires that it can be written with a leading term of \( \frac{n^{p+1}}{p+1}\) followed by arbitray terms in a polynomial. We've proven that for the \(p\)th term we will have the correct leading term followed by polynomials. Thus by induction we've proven that this argument is true.

Problem 8

Prove that every natural number is either even or odd.

Proof By Contradiction.

Definition of even is ...

\[E = \{n\in\mathbb{Z}: 2n\}\]

Definition of odd is...

\[O = \{n\in\mathbb{Z}: 2n+1\}\]

Assume \(\exists a \in \mathbb{N}: a \notin E \cup O\)

If \(a \notin E \cup O\) then \( a\notin \mathbb{Z}\) This because we know from the following proof

----------------------------------------------

Lemma 1 Proof by Induction

Base case

Let \(k_{base} = \{1,2\}\)

It is evident that \(k_{base} \subset E \cup O\) since \(k_1 \in O\) and \(k_2 \in E\) thus \(k_{base} \subset E \cup O\)

General case

Let the following set equal

\[k_n = \{-n,-n+1,...-1,0,1,...n-1,n\} \]

Assume that \(k_n \subset E \cup O \)

Induction

Let the following set equal

\[k_{n+1} = \{-n-1,-n,-n+1,...-1,0,1,...n-1,n, n+1\}\]

We can rewrite the set as the following.

\[k_{n+1} = k_n \cup \{-n-1, n+1 \}\]

We know that \( k_n \subset E \cup O\) we just need to check if \(\{-n-1, n+1 \} \subset E \cup O\)

Since we know that \(k_n \subset E \cup O \Rightarrow n \in O \) or \(n \in E\)

Case where \(n \in O\)

That means that \(n\) can be written as \(2k+1, \quad k\in \mathbb{Z}\) which is a definition of odd numbers. That would mean that \(n+1 = 2n+2 = 2(n+1)\). This is would always be even. Thus under this case \(n+1 \in E\)

We can rewrite \(-n-1 = -(n+1)\) in which case, \(-(n+1) = -(2k+2), \quad k\in \mathbb{Z}\) which is still an even number.

Case where \(n \in E\)

That means that \(n\) can be written as \(2k, \quad k\in \mathbb{Z}\) which is a definition of even numbers. That would mean that \(n+1 = 2k+1\). This is the definition of an odd number. Thus under this case \(n+1 \in O\)

We can rewrite \(-n-1 = -(n+1)\) in which case, \(-(n+1) = -(2k+1), \quad k\in \mathbb{Z}\) which is still an odd number.

Therefore \(\{-n-1, n+1 \} \subset E \cup O\).

We can extend this argument to span all integers for \(n\) and thus by induction we can prove that \(\mathbb{Z} \subseteq E \cup O\) is true.

----------------------------------------------

Back to the proof by contradiction

Since we know that \(\mathbb{Z} \subseteq E \cup O\). If \(a \notin E \cup O \Rightarrow a \notin \mathbb{Z}\). Since \(\mathbb{N} \subset \mathbb{Z} \Rightarrow a \notin \mathbb{N}\)

Thus this contradicts the assumption that \(a \in \mathbb{N}\) thus \(a \) must be even or odd to be a natural number.

Problem 17 a)

Prove that every composite number can be written as the product of prime numbers.

Proof By Induction.

Base Case

We know the first composite number thats not \(1\) is \(4\)

We can write \(4 = 2 \cdot 2\) which is a product of prime numbers, thus proven the base case.

General Case

Assume that for \(\forall n\in \mathbb{N}, n > 1 \) and any composite number \(< n\) can be written as the product of only prime numbers.

\(n\) can be written as \(n = a_1a_2\) assuming that \(a_1a_2 \in \mathbb{N}\).

Induction Case

We can test if \(n+1\) can be written as a product of prime numbers.

We can write \(n+1\) as a product \(n+1 = b_1b_2\) We know that \(b_1, b_2 < n+1 \) by themselves. Assuming that \(b_1,b_2 \in \mathbb{N}, b_1,b_2 \in [2, \infty)\)

We also know that \(b_1, b_2 < n \). Here's the proof.

----------------------------------------------

Lemma 1 Direct Proof

We can rewrite \(b_1 \) as \(\frac{n+1}{b_2}\)

If \(n - \frac{n+1}{b_2} > 0 \) then \(b_1, b_2 < n \)

\[n - \frac{n+1}{b_2} = \frac{b_2n - n-1}{b_2} \]

We know that the greatest lower bound for \(b_2\) is \(b_2=2\). We need to test for the greatest lower bound in order to minimize the expression.

\[n - \frac{n+1}{b_2} = \frac{2n - n-1}{b_2} \] \[n - \frac{n+1}{b_2} = \frac{n-1}{b_2} \]

We know every value is positive since \(n\) is composite and greater than 1. Thus its always true that

\[n - \frac{n+1}{b_2} > 0\]

Thus \(b_1, b_2 < n \)

Since \(b_1, b_2 < n \) by the assumption in the first case, any composite number \(< n\) can be written as the product of only prime numbers. Thus \(b_1,b_2\) can be written as a product of only prime numbers,\( \Rightarrow n+1\) can also be written as a product of only prime numbers.

Thus by induction every composite number can be written as the product of prime numbers.
