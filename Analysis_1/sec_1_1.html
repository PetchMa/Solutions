Problem 1.v

Prove: that \(x^n+y^n = (x-y) (x^{n-1} + n^{n-2}y+ ... + xy^{n-2}+y^{n-1})\)

We can rewrite it into summation notation. Let \(T= n-1\)

\[(x-y) (x^{n-1} + n^{n-2}y+ ... + xy^{n-2}+y^{n-1}) = (x-y) \sum_{k=0}^{T}x^{T-k}\cdot y^{k}\] \[ = \sum_{k=0}^{T} (x-y)x^{T-k}\cdot y^{k} \] \[ = \sum_{k=0}^{T} x^{T-k+1}\cdot y^{k} - x^{T-k}\cdot y^{k+1} \]

Notice that for the \(k\)th term the second term in the binomial is \(- x^{T-k}\cdot y^{k+1}\) which is the additive inverse for the first term of the \(k+1\) term which is \(x^{T-(k+1)+1}\cdot y^{k+1} =x^{T-k}\cdot y^{k+1}\). Thus every term in the series cancels out except for the first and last term, as there is no \(n+1\) term to cancel with nor is there a term before the start of the sequence for the first term to cancel. Thus we can collapse the summation notation into..

\[ = x^{T+1} - y^{T+1} \]

Sub back out value for \(T\)

\[\therefore (x-y) (x^{n-1} + n^{n-2}y+ ... + xy^{n-2}+y^{n-1})= x^{n} - y^{n} \]

Problem 7

Prove: the inequality which states the following. Assuming that \(0< a < b\)

\[a < \sqrt{ab} < \frac{a+b}{2} < b\]

Prove the first inequality. \(a < \sqrt{ab}\) We can square everything because we know \(0< a < b\) thus everything is positive and thus the inequality would still be true.

\[a^2 < ab\]

We know if the inequality is true, then \(ab-a^2 >0\) would be always positive.

\[ab-a^2 = a(b-a)\]

By definition \(0< a < b\) thus \(b-a > 0\) Thus \( a(b-a) >0 \Rightarrow a < \sqrt{ab}\)

Prove the second inequality. \(\sqrt{ab} < \frac{a+b}{2} \)

Similarly we can square everything and the inequality would hold since everything is positive. Thus if \((\frac{a+b}{2})^2 - ab >0 \Rightarrow \sqrt{ab} < \frac{a+b}{2}\)

\[(\frac{a+b}{2})^2 - ab = \frac{a^2+2ab+b^2}{4} - ab\] \[(\frac{a+b}{2})^2 - ab = \frac{a^2+2ab+b^2 -4ab}{4} \] \[(\frac{a+b}{2})^2 - ab = \frac{a^2-2ab+b^2 }{4} \] \[(\frac{a+b}{2})^2 - ab = \frac{(a-b)^2 }{4} \]

Notice the square, we've proven that \((\frac{a+b}{2})^2 - ab >0 \) thus we proven \( \sqrt{ab} < \frac{a+b}{2}\)

Prove the third inequality. \(\frac{a+b}{2} < b\)

If \(b - \frac{a+b}{2} > 0\) then \(\frac{a+b}{2} < b\)

\[b - \frac{a+b}{2} =\frac{2b -a-b}{2}\] \[b - \frac{a+b}{2} =\frac{b -a}{2}\]

By definition we know \(0< a < b\) thus \(b-a > 0 \Rightarrow b - \frac{a+b}{2} > 0 \) which proves \(\frac{a+b}{2} < b\)

We've proven \(a < \sqrt{ab} < \frac{a+b}{2} < b\) is true.

Problem 15

Prove: the inequality which states the following.

\[x^2 + xy + y^2 > 0,\]

and

\[x^4 +x^3y +x^2y^2 +xy^3 + y^4 > 0.\]

First Proof

Lets do the first one.

Case 1

If all values were positive. Then the entire thing would be positive, thus proven.

Case 2

If all values were negative. Then the entire equation would be positive again, because squares are always positive and the product of 2 negative numbers is always positive. Thus always positive.

Case 3

If values contained a mix of positive and negative, we can rewrite the equation as ...

\[x^2 - xy + y^2\]

Where \(x,y\) is the magnitude of the values.

Lets assume the case where \(x>y\) then we know that \(x^2>xy\) because, \(x^2-xy = x(x-y)\) and its always positive. This suggests that \(x^2 - xy + y^2\) is also positive since \(x^2>xy\)

If it were the reverse case were \(y> x\) then we know that \(y^2>xy\) because, \(y^2-xy = y(y-x)\) is always positive from definition. This suggests that \(x^2 - xy + y^2\) is also positive since \(y^2>xy\)

If \(x = y\) (the magnitude) we know that the equation would be\(x^2 - x^ + y^2 = y^2\) which we know is always positive since its a square.

Thus we've exhuasted all cases, therefore for any value of \(x,y\) we know that \(x^2 + xy + y^2 >0 \)

Second Proof

Lets exhuast all cases. We need to burrow our previous proof as a trick.

Case 1

If all values were positive. Then the entire thing would be positive, thus proven.

Case 2

If all values were negative. Then the entire equation would be positive again, because squares are always positive and the product of 2 negative numbers is always positive. Thus always positive.

Case 3

If values contained a mix of positive and negative, we can rewrite the equation as ...

\[(x^4 +x^2y^2 + y^4) -xy^3 - x^3y \]

Which can be written in a more suggestive way.

\[(x^4 +x^2y^2 - x^3y) +(y^4 -xy^3) \]

Where \(x,y\) is the magnitude of the values. We know everything in the bracket is positive from initial observations.

Lets assume the case where \(y>x\) then we know that \(y^4>xy^3\) because, \(y^4-xy^3 = y^3(y-x)\) and its always positive based on definition. We also know that \(x^4+x^2y^2 >x^3y\) since \(x^4+x^2y^2 - x^3y\) is always positive. This is because we can rewrite the equation as \(x^2(x^2+y^2 - xy)\) which we've proved above that \(x^2+y^2 - xy >0\) if \(x,y\) are magnitudes. Thus each term is positive.

\[\therefore x^4 +x^3y +x^2y^2 +xy^3 + y^4 > 0.\]

Problem 19

Prove: the Schwarz inequality which states the following.

\[x_1y_1 + x_2y_2 \leq \sqrt{x_1^2+x_2^2} \sqrt{y_1^2+y_2^2}\]

Assuming that \(x_1y_1 + x_2y_2\) is positive, we can square everything and the inequality would still be true.

The case where \(x_1y_1 + x_2y_2 < 0\) the condition would automatically be satisfied since the right hand side is ALWAYS positive. Thus always greater than.

Lets play with this inequality instead.

\[(x_1y_1 + x_2y_2)^2 \leq (x_1^2+x_2^2)(y_1^2+y_2^2)\]

If the inequality is true then \((x_1^2+x_2^2)(y_1^2+y_2^2) - (x_1y_1 + x_2y_2)^2 > 0 \) is always positive. We expand

\[(x_1^2+x_2^2)(y_1^2+y_2^2) - (x_1y_1 + x_2y_2)^2 = (x_1^2y_1^2+y_2^2x_1^2+x_2^2y_1^2+x_2^2y_2^2) - (x_1^2y_1^2 +2x_1y_1 x_2y_2 + x_2^2y_2^2)\]

Subtract like terms.

\[ = (y_2^2x_1^2+x_2^2y_1^2) - (2x_1y_1 x_2y_2 )\]

We can bring in the value into the bracket.

\[ = y_2^2x_1^2-2x_1y_1 x_2y_2 +x_2^2y_1^2\]

We can factor! This is a perfect square.

\[ = (y_2x_1-x_2y_1)^2\]

We know that perfect squares are always positive. Thus \((x_1^2+x_2^2)(y_1^2+y_2^2) - (x_1y_1 + x_2y_2)^2 > 0 \) is true! The only case in which they would equal, the zero case, is when \(y_2 = y_1\) and \(x_2 = x_1\) when the square equals 0.
