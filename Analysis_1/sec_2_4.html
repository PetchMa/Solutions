Problem 14

Describe the function \(g\) in terms of \(f\)

\(g(x) = f(x)+c\)

Vertical shift by \(c\) units up.

\(g(x) = f(x+c)\)

Horizontal shift by \(c\) units to the left.

\(g(x) = cf(x)\)

Vertical stretch by a factor of \(c\)

\(g(x) = f(cx)\)

Horizontal compression by a factor of \(c\)

\(g(x) = f(\frac{1}{x})\)

Everything that happens far out happens near \(0\) and everything that happens near \(0\) happens far out now.

\(g(x) = f(|x|)\)

Turns any function into an even function. Also it creates a reflection along the y axis.

\(g(x) = |f(x)|\)

Reflection on the x axis for negative values of \(f(x)\)

\(g(x) = min(f(x), 0)\)

Retains all the negative values, and for any positive values its 0.

Problem 22

Show that the square of the distance from \((c, d)\) to \((x, mx)\) is

\[x^2(mx^2+1)+x(-2md-2c)+d^2+c^2\]

We can use the distance formula.

\[d(x,y)^2 = (x_1-x)^2 + (y_1-y)^2\]

We replace the variables

\[d(x,mx)^2 = (c-x)^2 + (d-mx)^2\] \[d(x,mx)^2 = c^2-2cx + x^2 + d^2-2dmx+m^2x^2\] \[d(x,mx)^2 = c^2+(-2cx-2dmx) + x^2 + d^2+m^2x^2\] \[d(x,mx)^2 = x^2(m^2+1)+ x(-2c-2dm) + d^2+c^2\]

Find the minimum distance.

Notice that this is in quadratic form and we can solve for the mimimum of the distance by locating the vertex.

\[y = \frac{-b^2}{4a}+c\]

Thus we plug the coefficents into the formula.

\[d(x,mx)^2 =\frac{-(-2c-2dm)^2}{4(m^2+1)}+d^2+c^2 \] \[d(x,mx)^2 =\frac{-(4m^2d^2 +8mdc+4c^2)}{4(m^2+1)}+d^2+c^2 \] \[d(x,mx)^2 =\frac{(-m^2d^2 -2mdc-c^2) - d^2m^2+d^2+c^2m^2+c^2}{(m^2+1)}\] \[d(x,mx)^2 =\frac{ -2mdc +d^2+c^2m^2+c^2}{(m^2+1)}\] \[d(x,mx)^2 =\frac{(d-cm)^2}{(m^2+1)}\]

Since we want mimum distance we take the square root. But since we're square rooting a already perfect square we can add in absolute value bars and take out the square.

\[d(x,mx) =\frac{|d-cm|}{\sqrt{m^2+1}}\]

What is the shortest distance if it was between \((c,d)\) and \((x, mx+b)\)??

We use the exact same technique as we did before.

\[d(x,y)^2 = (x_1-x)^2 + (y_1-y)^2\]

We replace the variables

\[d(x,mx+b)^2 = (c-x)^2 + (d-(mx+b))^2\] \[d(x,mx+b)^2 = (c^2-2cx+x^2) + (d^2-dmx-db-dmx+m^2x^2+mxb-db+bmx+b^2)\]

We can simplify

\[d(x,mx+b)^2 = m^2x^2+2bmx-2dmx+b^2-2ba+c^2-2cx+d^2+x^2\] \[d(x,mx+b)^2 = x^2(m^2+1)+x(2bm-2dm-2c)+(b^2+c^2+d^2-2bd)\]

We know that this is in quadratic form \(ax^2+bx+c\) and that the formula for the vertex (ie critical point on the graph is the following.)

\[y = \frac{-b^2}{4a}+c\]

Thus we plug the coefficents into the formula.

\[d(x,mx+b)^2 =\frac{-(2bm-2dm-2c)^2}{4(m^2+1)}+(b^2+c^2+d^2-2bd) \] \[d(x,mx+b)^2 =\frac{4c^2m^2+8bcm−8cdm+4b^2−8bd+4d^2}{4(m^2+1)} \] \[d(x,mx+b)^2 =\frac{c^2m^2+2bcm−2cdm+b^2−2bd+d^2}{(m^2+1)} \]

The minimum distance is thus the following after factoring.

\[d(x,mx+b) =\sqrt{\frac{(cm-d)^2+2bcm+b^2−2bd}{(m^2+1)} }\]
