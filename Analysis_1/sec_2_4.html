Problem 22

Show that the square of the distance from \((c, d)\) to \((x, mx)\) is

\[x^2(mx^2+1)+x(-2md-2c)+d^2+c^2\]

We can use the distance formula.

\[d(x,y)^2 = (x_1-x)^2 + (y_1-y)^2\]

We replace the variables

\[d(x,mx)^2 = (c-x)^2 + (d-mx)^2\] \[d(x,mx)^2 = c^2-2cx + x^2 + d^2-2dmx+m^2x^2\] \[d(x,mx)^2 = c^2+(-2cx-2dmx) + x^2 + d^2+m^2x^2\] \[d(x,mx)^2 = x^2(m^2+1)+ x(-2c-2dm) + d^2+c^2\]

Find the minimum distance.

Notice that this is in quadratic form and we can solve for the mimimum of the distance by locating the vertex.

\[y = \frac{-b^2}{4a}+c\]

Thus we plug the coefficents into the formula.

\[d(x,mx)^2 =\frac{-(-2c-2dm)^2}{4(m^2+1)}+d^2+c^2 \] \[d(x,mx)^2 =\frac{-(4m^2d^2 +8mdc+4c^2)}{4(m^2+1)}+d^2+c^2 \] \[d(x,mx)^2 =\frac{-m^2d^2 -2mdc-c^2) - d^2m^2+d^2+c^2m^2+c^2}{(m^2+1)}\] \[d(x,mx)^2 =\frac{ -2mdc +d^2+c^2m^2+c^2}{(m^2+1)}\] \[d(x,mx)^2 =\frac{(d-cm)^2}{(m^2+1)}\]

Since we want mimum distance we take the square root. But since we're square rooting a already perfect square we can add in absolute value bars and take out the square.

\[d(x,mx) =\frac{|d-cm|}{\sqrt{m^2+1}}\]
