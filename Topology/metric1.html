The class is introduced by connecting the ideas in Analysis and generalizing them with these arbitrary sets in a very broad topic of point set topology.

Lets say we're given som arbitrary space which is a collection of "stuff" and this is what we call a set which is described as: \(X\). With this space \(X\) we wish to add addition structure, this structure is called a Metric which we define as \(d(x,y)\) where \(x,y\in X\). You can think of this as some general idea of a Distance

Metric Definition:

  1. Function is definite: \(d(y,x) = 0 \) if and only if \(x=y\)
  2. Function commutes: \(d(y,x) = d(y,x) \)
  3. Obeys triangle inequality commutes: \(d(y,x) \leq d(x,z)+ d(z,y) \)

Example of Metrics

Standard Euclidean Metric:

\[d(x,y) = \sqrt{\sum_{i=1}^n (x_n-y_n)^2}\]

Discrete Metric

\[d(x,y) =\begin{cases} 0, & x=y \\ 1, & x\neq y \end{cases}\]

We also have the space of continous functions between \((a,b)\) which is defined as:

\[d(f,g) = \sqrt{\int_a^b (f(x)-g(x))^2 dx}\]

Note On Spaces

There are various other canonical spaces one of them being the hilbert space or \(\ell^p\) space. These are defined as the following:

  1. \(\ell^1\) space These are the space of all sequences who's series is absolutely convergent. Defined with the following metric:

    \[||x|| := \sum^\infty |x_n| \leq \infty\]

  2. \(\ell^2\) space the space of square-summable sequences, which is a Hilbert space

    \[||y||_2 := \sqrt{\sum^\infty y_n^2} \leq \infty\]

  3. \(\ell^p\) space the space of square-summable sequences, which is a Hilbert space

    \[||y||_p := \sqrt[p]{\sum^\infty y_n^p} \leq \infty\]

  4. \(\ell^\infty\) space the space of bounded sequences.

    \[||z||_\infty := \text{sup}_{n\in \mathbb{Z}} |z_n| \leq \infty\]

Continuity

We can devise a means of defining a limit in these for functions between metric spaces using the definition of the metric. This follows exactly from Analysis of \(\epsilon-\delta\) definitions except we use a metric:

\[\forall \epsilon > 0 \exists \delta >0 : d(x,y)< \delta \Rightarrow d(f(x),y(x)) < \epsilon \quad \text{for }\mathbb{R}^n \]

Open Balls

Imagine a point in some arbitrary space, since we defined the notion of a distance we can then "sweep" around some abstract shape of a sphere which can be called \(B_a(r)\) where \(a\in X\) and \(r\) is some radius. We get:

\[B_a(r) := \{\forall x \in X | d(x,a) < r\}\]

Continuity - In Terms of Open Balls

The idea is that the mapping of an entire ball to the image and then taking the preimage, if that preimage of that map of the ball is a subset of the original ball, then the map is continous. This is determined by the \(\delta\) and the \(\epsilon\) which is the radius.

\[\forall \epsilon >0 \exists \delta >0 : f(B_a(\delta) \subset B_{f(a)}(\epsilon)\]

The preimage defintion is:

\[ a \in B_a(\delta) \subset f^{-1}(B_{f(a)}(\epsilon))\]

Definition

Open set \(A\subseteq M\) is Open in the metric space \(M\) if it is a union of open balls.

The set of all open sets in the metric space \((M,d)\) is called a metric topology on \((M,d)\) which we call \(T(M,d)\)

 

We can then formulate continuity in terms of open sets:

 

Theorem

\(f: (M_1,d_1) \to (M_2,d_2)\) is continous if and only if \(U \in T (M_1,d_1) \Rightarrow f^{-1}(U) \in T(M_1,d_1)\).

This is the same as the definition above since the topology is a power set of the metric space and so if says for ALL open sets this rule is obeyed.

Limit Point

We say that \(a\) is a limit point of \(X\) if the open ball around the point has some elements in \(X\) in other words for any open ball centered at \(a\) we have that \(B_a(r) - \{a\} \cap X \neq \emptyset\)

The limit around it is called \(\text{lim}(X), X'\)

Boundary Point

The boundary point is the edge of a bound. Imagine taking the edge point and constructing a open ball around it, for any open ball centered at that point there are elements in the set AND out of the set. Thus the definition of boundary denoted by \(\partial X\) defined by

\[\forall \epsilon > 0 : \begin{cases} B_a(\epsilon) \cap A \neq \emptyset \\ B_a(\epsilon) \cap A^c \neq \emptyset \\ \end{cases}\]

We can piece together the entire space by:

\[M = \text{int}(X) \sqcup \partial(X) \sqcup\text{cl}(X) \]

Or it can be described as the union of boundary and the interior. \(\overline{A} = A \cup \partial A\)

Bounding Spaces

We can define the "diameter" of a some subspace of \(\mathbb{R}^n\) where it is the maximum distance between any point in the subspace.

\[\text{diam}(C) = \text{sup}_{x,y\in C}d(x,y)\]

We can also define the distance between subspaces as the smallest distance.

\(d(A,B) = \text{inf}_{x\in A, y \in B}d(x,y)\)

Definition

We define that \(C \subset \mathbb{R}\) is compact if and only if for any arbitrary covering for \(C\) there is a fininte number of subcovers.

Formally we write:

\[C \subset \bigcup_{k=1}^n B_{a_k}(r_k)\]

Here are a number of equivalent definitions:

Compactness

Compactness if and only if for any arbitrary covering for \(C\) there is a fininte number of subcovers.

Countably Compactness for every infinite subset of \(M\) there is a limit point.

Sequentially Compactness for every sequence in \(M\) has a convergent subsequence

Complete and Totally Bounded every cauchy sequence converges and is totally bounded. Specifically \(\forall \epsilon >0 \exists \) a finite number of open balls of radius \(\epsilon\) that covers \(M\).

Complete is a more general class of compactness.

Dense

We can call something is Dense if \(\overline X = M\). Intuitively this means that you can never find a two points such that there doesnt exist an open ball where they intersect each other. In other words there is always something in between.

Nowhere Dense

This makes intuitive sense because for any point in the set you can always find an exact region where something "ends" then clearly its not dense. If it were dense then you shouldn't be able to find this "boundary".

Category of Metric Spaces

We know that any compact metric space is complete. There are non-compact metric spaces which are also complete. The calss of complete metric spaces is much more general and this crucial property is: Baire category theorem.

Definition

A subset \(X\) of the metric space is said to be the first category if it is a countable union of nowhere dense sets. Otherwise, \((M,d)\) is said to be secound category.

Baire Category Theorem

All complete metric space is of the second category.

Here are some supposedly useful applications of this theorem:

Theorem

Let \(M,d\) be a complete metric space. Suppose that \(\{U_n\}_{n\geq 1}\) is a countable family of open dense subsets in \(M\). Then \(\cup_{n=1}^\infty U_n \neq \emptyset\)

Theorem: Uniform Boundedness Principle

Let this be a complete metric space. Suppose that \(\mathcal{F}\) is a family of continous real-valued functions on the metric space such that:

\[\forall a \in M \exists C_a \in \mathbb{R} : f \in \mathcal{F} \Rightarrow |f(a)| \leq C_a\]

Then there exists a non-empty open set \(U \subseteq M\) and a constant \(C\) such that \(f\in\mathbb{F}, x\ in U \rightarrow |f(x)|\leq C\)
