Remainder: Metric Spaces

If we say that \(X\) is a subset of the metric space \((M,d)\)

then \((X,d_x)\) is also a metric subspace and the Induced metric topology is \(T(X,d_x\).

We say that two metrics \(d, D\) are the same on the space \(M\)

  • Strongly Equivalent: \[C_1d(x,y) \leq D(x,y) \leq C_2d(x,y), \forall x,y\in M\]
  • Topologically Equivalent: \[T(M,d) = T(M,D)\]

    • Definition: Topology

    A subset \(\tau = \tau_x\) of \(\mathcal{P}(X)\) is called a topology on \(X\) if it satisfies these conditions:

    1. \(\emptyset\) and \(X\) are in \(\tau\)
    2. If \(U_1,U_2 \in \tau\) then \(U_1 \cap U_2 \in \tau\)
    3. If \(U_\alpha \in \tau\) for all \(\alpha \in A\) then \(\bigcup_{\alpha \in A}U_\alpha \in tau\)

    The elements of \(\tau \) are open sets on \(X\) and the pair \((X,\tau)\) is called the topological space. Typically we say ommit the \(\tau\).

    Some examples, we see that \(\{\emptyset,X\}\) and \(\mathcal{P}(x)\) are topologies on \(X\). The first one being the trivial topology called \(\tau_a\) and the second being the discrete topology called \(\tau_b\)

    We also know that \(\mathcal{P}(x) = T(X,d_{discrete})\), now the question is for any topological space is what if \(\tau = T(X,d)\)? If such a \(d\) on \(X\) exists, then the topological space is called metrizable topological space.

    Continous Maps

    Continous if the preimage of any open set is an open set. Or if its closed then the preimage is closed.

    Open if the image of an open is open. Closed if any image of closed set is also closed.

    Homeomorphism: if it is a bijection, which is continous and open at the same time. Then \(f\) is also a closed map.

    Definition: Induced Topology

    For a subset \(Y\) of a topological space \((X,\tau)\) we define the induced topology on \(Y\) to be the subset \(\tau_Y\in \mathcal{P}(Y) \) such that: \[A\in \tau_Y \Leftrightarrow A = U\cap Y, U\in \tau\]

    We want a new definition for open sets: we define it as:

    For a subset \(Y\) of a topological space \(X\) define befine its Open neighborhood to be an open set containing \(Y\)

    Interior: the maximal open subset of \(X\) contained in \(Y\) \(\Leftrightarrow \) the union of all opensubsets of \(Y\).

    Isolated points \(Y^{iso}\): the subset of \(Y\) such that for each of its points there exists an open neighborhood which contains no other points from \(Y\).

    Limit points \(Y^{lim} = Y'\): the subset of \(X\) such that for each of its points \(a\) any open neighborhood of \(a\) contains at least one point of \(Y\) different from \(a\).

    closure \(cl(Y) = \overline{Y}\) defined as \(Y\cup Y^{lim} = Y^{iso}\sqcup Y^{lim}\)

    Exterior is the maximal openset of \(X\) contained in the compliment of \(Y\). Same as the union of all open sibsets of \(Y^C\) which is the maximal subset.

    Boundary \(\partial (Y)\) for each point of \(a\) such that the open neighborhood has both a point inside and a point outside.

    Dense we will define as \(\overline{Y}=X\). The closure of a subset os the entire space. Any open set of \(X\) it exists a point from \(Y\).

    Nowhere Dense if \(\text{int}(\overline{Y})= \emptyset\) that is \(X =X\ \emptyset\).

    \(\tau_1\) is coarser/weaker/smaller than \(\tau_2\) if \(\tau_1\subseteq \tau_2\). Finer or stronger if its the other subset direction.

    After defining all of this we can also find the smallest topology or the coarsest topology.

    \(T\langle S \rangle\) equals the intersection of all topologies on \(X\) containing all elements from \(S\).

    Definition: Subbasis

    A subset \(S\) in \(\mathcal{P}(X)\) is called a subbasis for \(\tau\) if \(\tau = T\langle S\ rangle\). In this case, we can say that \(\tau\) is generated by \(S\) thus \(S =S(\tau)\)

    We can introcue a topology on the product set \(\prod X\).

    First we can consider the set of all projection maps: \(\{\pi_\alpha: X\to X_\alpha\}_{\alpha\in A}\) and the set \(S=\{\pi^{-1}_\alpha(U_\alpha) | U_{\alpha}\in \tau_{\alpha},\alpha \in A\}\).

    The topology \(T\langle S \rangle \) is generated by \(S\) is called the product topology. \(\tau = \prod_{\alpha \in A}\tau_\alpha\)

    We say that \(B\subseteq \tau\) is a basis for \(\tau\) if an open set of \(X\) is a union of elements from \(B\) in this case we can write \(B = B(\tau)\) and that \(\tau = T(B)\).

    Theorem

    There exists a topology \(\tau\) on \(X\) such that \(\tau = T(B) \) and \(B = B(\tau)\) if and ONLY if \(X = \bigcup B\) for any \(B_1, B_2 \in B:\)

    \[a \in B_1 \cap B_2 \Rightarrow \exists B(a) \in B:a\in B(a) \subseteq B_1 \cap B_2\]

    Prove Primes are Infinite

    We can prove cool things now. For example the infinite primes!!!

    Let \(X = \mathbb{Z}\) be the set of integers and let \(B\) be the set of all bi-infinit arithmetic progressions then \(B \in B(X)\).

    Indeed then \(\mathbb{Z} = p\mathbb{Z}\cup \{1+\mathbb{Z}\}\cup \cdots \{(p-1)+p\mathbb{Z}\}\) and if \(a\in \{a_1+nd_1| n\in \mathbb{Z}\}\cap \{a_2 + nd_2|n\in \mathbb{Z}\}\), then \[a\in \{a+nd_1d_2| n\in \mathbb{Z}\} \subseteq \{a_1+nd_1|n\in \mathbb{Z}\} \cap \{a_2+nd_2|n\mathbb{Z}\}\]

    Let us consider the topology \(\tau\) on \(Z\) with the basis \(B\). Observe that for any \(p\in \mathbb{P}\) the set \(p\mathbb{Z}\) is both open and closed in \(\tau\). We know for sure because the compliment \((p\mathbb{Z})^C\) is a union of \(p-1\) arithmetic progressions, corresponding to the non-zero remainders \(\text{mod }p\) and hence is open in \(\tau\).

    Now suppose there were only finitely many primes. It implies that the set \(\bigcup_{p\in \mathbb{P}}p\mathbb{Z})\) would be closed. However the compliment of this set in \(X\) is a two element set \(1,-1\) which is not open! Indeed it can not contain any bi-infinite arithmetic progression since it is finite. Contradiction.
