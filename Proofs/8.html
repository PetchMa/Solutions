Problem 1

Prove: that the function \(f(x)=x\) is not a periodic function.

Proof By Contradiction

We defined a periodic function to have a period of \(T\) that is \(\exists T, T\neq 0, T \in \mathbb{R}\) such that \(\forall x \in \mathbb{R}, \quad f(x+T) = f(x)\). In other words, if we add a period to the input we should get the same number.

Assume: that the function \(f(x)\) is periodic.

If \(f(x)\) is periodic then \(f(x+T) = f(x) \Rightarrow f(x+T) - f(x) = 0\)

\[f(x+T) - f(x) = (x+T) - x = T\]

In order to satisfy the definition of a period, \(T= 0 \)

By definition of the periodic function \(T \neq 0\), thus contradicting the definition of a periodic function, suggesting that \(f\) is not periodic.

Problem 3

Compute the following:

\(\cos(\arctan(x))\)

We can solve this using algebra. We know this is true:

\[\cos^2(x)+ \sin^2(x) = 1\]

The angles can be any angle correct?Lets sub in \(\arctan(x)\)

\[\cos^2(\arctan(x))+ \sin^2(\arctan(x)) = 1\] \[\frac{\cos^2(\arctan(x))}{\cos^2(\arctan(x))}+\frac{\sin^2(\arctan(x))}{\cos^2(\arctan(x))} = \frac{1}{\cos^2(\arctan(x))}\] \[1+\tan^2(\arctan(x)) = \cos^{-2}(\arctan(x))\] \[1+(\tan(\arctan(x)))^2 = \cos^{-2}(\arctan(x))\]

taking the inverse of the original function gives back \(x\)

\[1+(x)^2 = \cos^{-2}(\arctan(x))\]

We re-arrange for \(\cos(\arctan(x))\)

\[\frac{1}{\sqrt{1+(x)^2} }= \cos(\arctan(x))\]

We can use any easier trick called thes triangle method. By rewriting the \(arctan(x)\) as the angle in a triangle. We know that \(\tan\) is opposite over adjacent. We can get the hypotenuse from pythagorean theorem.

We know what the \(\arctan(x)\) is, we can just copy off the values for \(\cos(\arctan(x))\) which is adjacent over hypotenuse.

\[\cos(\arctan(x)) = \frac{1}{\sqrt{x^2+1}}\]
