Proofs By Induction

Problem 1

Prove the following.

\[\sum_{k=1}^{n}\frac{1}{k(k+1)} = \frac{n}{n+1}\]

Proof by induction goes the following: Prove the base case is true. Assume the general case is true, and then prove that the general case + 1 is true. Thus by induction you can prove the entire chain up untill the general case is true.

Let \(S(n) = \sum_{k=1}^{n}\frac{1}{k(k+1)}\) and let \(S'(n) = \frac{n}{n+1}\). Prove that \(S(n)=S'(n)\)

1- Base Case where \(n=1\)

We prove that \(S'(1) = S(1)\). If that were true, then \(S(n)-S'(n) = 0\)

\[S(n)-S'(n) = [\sum_{k=1}^{n}\frac{1}{k(k+1)}] - \frac{n}{n+1} = \] \[= \frac{1}{1(1+1)} - \frac{1}{1+1}\] \[= 0\] \[\therefore S(1) = S'(1)\]

2- Assume General Case Is True.

We assume that the following case is true for \(n\)

\[S(n) = \sum_{k=1}^{n}\frac{1}{k(k+1)} = \frac{n}{n+1} = S'(n)\]

3- Prove That Every Sequential \(n+1\) is also true.

If \(S(n+1) - S'(n+1) = 0\) then \(S(n+1) = S'(n+1)\)

\[S(n+1) - S'(n+1) = [\sum_{k=1}^{n+1}\frac{1}{k(k+1)}] - \frac{n+1}{n+2}\]

We know that the series \(S(n+1) = S(n) +\frac{1}{(n+1)((n+1)+1)} \) correct?

\[S(n+1) - S'(n+1) = [S(n) +\frac{1}{(n+1)((n+1)+1)} ] - \frac{n+1}{n+2}\]

We've already assumed that \(S(n) = S'(n) = \frac{n}{n+1} \)

\[S(n+1) - S'(n+1) = [\frac{n}{n+1} +\frac{1}{(n+1)(n+2)} ] - \frac{n+1}{n+2}\] \[S(n+1) - S'(n+1) = [\frac{n(n+1)(n+2)}{(n+1)(n+1)(n+2)} +\frac{n+1}{(n+1)(n+1)(n+2)} ] - \frac{n+1}{n+2}\] \[S(n+1) - S'(n+1) = [\frac{n(n+1)(n+2) + (n+1)}{(n+1)^2(n+2)} ] - \frac{n+1}{n+2}\] \[S(n+1) - S'(n+1) = [\frac{n(n+2) + 1}{(n+1)(n+2)} ] - \frac{n+1}{n+2}\] \[S(n+1) - S'(n+1) = [\frac{n^2+2n+ 1}{(n+1)(n+2)} ] - \frac{n+1}{n+2}\] \[S(n+1) - S'(n+1) = [\frac{(n+1)^2}{(n+1)(n+2)} ] - \frac{n+1}{n+2}\] \[S(n+1) - S'(n+1) = [\frac{(n+1)}{(n+2)} ] - \frac{n+1}{n+2}\] \[S(n+1) - S'(n+1) = 0\] \[\therefore S(n+1) = S'(n+1) \]

Thus by induction we have proven that \(\sum_{k=1}^{n}\frac{1}{k(k+1)} = \frac{n}{n+1}\).

The reason why this works is because, although we've assumed the general case \(n\) works. We've proven that every successive term afterwards is true if the general case is true. But since we've proven that the base case works, we can replace the general case with the base case and say that since \(n=1\) works, we know then that the next case \(n=2\) would also work. And since we've proven that \(n=2\) works we can say \(n=3\) also works .... and so on and so forth all the way up until \(n\).

Problem 2

Prove that the function \(n^2+3n+2\) where \(n \in \mathbb{N}\) is always even.

Proof by Induction.

Let \(p(n) = n^2+3n+2\)

1- Base Case where \(n=1\)

We can show the base case is even and thus true.

\(p(1) = 1^2+3(1)+2 = 6\)

2- Assume General Case Is True.

We assume that the following case is true for \(n\), that \(p(n)\) is even.

\[p(n) = n^2+3n+2\]

3- Prove That Every Sequential \(n+1\) is also true.

We can subsitute.

\[p(n+1) = (n+1)^2+3(n+1)+2\] \[p(n+1) = n^2+2n+1+3n+3+2\] \[p(n+1) = n^2+5n+6\] \[p(n+1) = (n+2)(n+3)\]

Since \(p(n) = n^2+3n+2 = (n+2)(n+1)\) is even. We know that either \((n+2)\) or \((n+1)\) must be even.

Case where \((n+2)\) is even, then \(p(n+1)\) must also be even since its a common factor.

Case where \((n+1)\) is even, then \(p(n+1)\) must also be even since the factor \((n+3) = (n+1)+2\) and if \((n+1)\) is even then so is \((n+1)+2 \Rightarrow (n+3)\) is also even. Thus \(p(n+1)\) is even.

Thus by induction we have proven that \(p(n)\) is always even.

Problem 3

Prove the following:

\[\sum_{k=1}^{n}k^2 = \frac{1}{6}n(n+1)(2n+1) \]

Let \(S(n) = \sum_{k=1}^{n}k^2\) and let \(S'(n) = \frac{1}{6}n(n+1)(2n+1)\). Prove that \(S(n)=S'(n)\)

1- Base Case where \(n=1\)

If \(S(1)-S'(1) = 0\) then \(S(1) = S'(1) \)

\[S(1)-S'(1)=[\sum_{k=1}^{n}k^2] - \frac{1}{6}n(n+1)(2n+1) \] \[= 1^2 -\frac{6}{6}\] \[= 0 \] \[\therefore S(1)=S'(1)\]

The base case is true.

2- Assume General Case Is True.

We assume that the following case is true for \(n\)

\[S(n) = \sum_{k=1}^{n}k^2 = \frac{1}{6}n(n+1)(2n+1) \]

3- Prove That Every Sequential \(n+1\) is also true.

If \(S(n+1)-S'(n+1) = 0\) then \(S(n+1) = S'(n+1) \). We can sub the values in.

\[S(n+1)-S'(n+1)=[\sum_{k=1}^{n+1}k^2] - \frac{1}{6}(n+1)(n+2)(2n+3) \]

We know that the series \(S(n+1) = S(n) +(n+1)^2 \) correct?

\[S(n+1)-S'(n+1)=S(n) +(n+1)^2 - \frac{1}{6}(n+1)(n+2)(2n+3) \]

We already assumed that \(S(n) = \frac{1}{6}n(n+1)(2n+1)\)

\[S(n+1)-S'(n+1)=\frac{1}{6}n(n+1)(2n+1) +(n+1)^2 - \frac{1}{6}(n+1)(n+2)(2n+3) \] \[S(n+1)-S'(n+1)=\frac{1}{6}n(n+1)(2n+1) +(n+1)^2 - \frac{1}{6}( 2 n^3 + 9 n^2 + 13 n +6 ) \] \[S(n+1)-S'(n+1)=\frac{1}{6}(2 n^3+ 3n^2 + n ) +(n^2+2n+1) - \frac{1}{6}( 2 n^3 + 9 n^2 + 13 n +6 ) \] \[S(n+1)-S'(n+1)=\frac{1}{6}(2 n^3+ 3n^2 + n ) + \frac{1}{6}(6n^2+12n+6) - \frac{1}{6}( 2 n^3 + 9 n^2 + 13 n +6 ) \] \[S(n+1)-S'(n+1)=\frac{1}{6}(2 n^3 + 9 n^2 + 13 n +6 ) - \frac{1}{6}( 2 n^3 + 9 n^2 + 13 n +6 ) \] \[S(n+1)-S'(n+1)=0 \] \[\therefore S(n)=S'(n)\]

Thus by induction we have proven that \(\sum_{k=1}^{n}k^2 = \frac{1}{6}n(n+1)(2n+1)\).
