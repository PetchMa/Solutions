Prove the following:

\[∀a, b ∈ F, a · b = 0 ⇒ a = 0 \quad or \quad b = 0\]

Notice that this has 4 total cases since its an or operation. Truth tables. But we know that \(a\cdot b\) must equal to 0, in this situation, we just need to prove the contrary is false. Where \(a,b \neq 0, \quad a\cdot b = 0\)

Proof by Contradiction.

Assume that \(a,b \neq 0, \quad a\cdot b = 0\)

\[ 0 = 0 \cdot b^{-1}, \quad assumption\] \[ 0 = (a\cdot b) \cdot b^{-1}, \quad assumption\] \[ 0 = a\cdot (b \cdot b^{-1}), \quad M1\] \[ 0 = a\cdot (1), \quad M1\] \[ 0 = a, \quad M4\] \[ 0 = a\]

This contradicts \(a\neq 0\) and thus the assumption is false.

Since given \(a, b \neq 0, \quad a \cdot b \neq 0\) we know that \(a\cdot b = 0\) must be true. Then the only option left is that \(a\) or \(b\) equal to \(0\) that can result in \(ab=0\)
