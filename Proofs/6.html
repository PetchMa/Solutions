Problem 1

Prove: that the composition of two decreasing functions is increasing.

Let \(f: X \rightarrow Y\) and \(x_1 \leq x_2 \Rightarrow f(x_1) \geq f(x_2)\). This means that for every \(y\) greater than \(x\) the function value \(f(y)\)is less than \(f(x)\). Thus \(f(x)\) is decreasing.

Let \(g: Y \rightarrow Z\) and \(y_1 \leq y_2 \Rightarrow g(y_1) \geq g(y_2)\).

We define the composite function as \(g(f(x))\).

We know that by definition that \(x_1 \leq x_2 \Rightarrow f(x_1) \geq f(x_2)\). We know that then the relationship \(f(x_2) \leq f(x_1) \Rightarrow g(f(x_2)) \geq g(f(x_1))\) by definition of a decreasing function.

Since we definied that \(x_1 \leq x_2 \) we know then that \( g(f(x_1)) \leq g(f(x_2)) \) which is the definition of an increasing function.

Problem 2

Prove: the composition of a decreasing function with an increasing function is decreasing (regardless of order)

Let \(f: X \rightarrow Y\) and \(x_1 \leq x_2 \Rightarrow f(x_1) \leq f(x_2)\). This means that for every \(y\) greater than \(x\) the function value \(f(y)\)is greater than \(f(x)\). Thus \(f(x)\) is increasing.

Let \(g: Y \rightarrow Z\) and \(y_1 \leq y_2 \Rightarrow g(y_1) \geq g(y_2)\). Decreasing function.

We define the composite function as \(g(f(x))\).

By definition, \(x_1 \leq x_2 \Rightarrow f(x_1) \leq f(x_2)\) and thus by the second definition \(f(x_1) \leq f(x_2) \Rightarrow g(f(x_1)) \geq g(f(x_2))\) Thus we shown that \(x_1 \leq x_2 \Rightarrow g(f(x_1)) \geq g(f(x_2))\) which means its a decreasing function.

Similarly if we were to flip the order...

By definition, \(x_1 \leq x_2 \Rightarrow g(x_1) \geq g(x_2)\) and thus by the second definition \(g(x_2) \geq g(x_1) \Rightarrow f(g(x_2)) \leq f(g(x_1))\) Thus we shown that \(x_1 \leq x_2 \Rightarrow g(f(x_1)) \geq g (f(x_2))\) which means its a decreasing function.

Problem 4

Prove: Prove that if \(f\) is both increasing and decreasing then \(f\) is a constant function.

Define \(f\) as decreasing first.

Let \(f: X \rightarrow Y\) and \(x_1 \leq x_2 \Rightarrow f(x_1) \geq f(x_2)\).

Define \(f\) as increasing.

Let \(f: X \rightarrow Y\) and \(x_1 \leq x_2 \Rightarrow f(x_1) \leq f(x_2)\).

The only case where \(f(x_1) \geq f(x_2)\) and \(f(x_1) \leq f(x_2)\) to be both satisfied is when \(f(x_1) = f(x_2)\)

Thus we define \(f\) as \(x_1 \leq x_2 \Rightarrow f(x_1) = f(x_2)\) which is the definition of a constant function.

Prove that \(f : \mathbb{R} → \mathbb{R}, f(x) = -x\) is strictly decreasing

If \(x_1 < x_2 \Rightarrow f(x_1) > f(x_2)\) then \(f\) is strictly decreasing

Assuming that \(x_1 < x_2\) we know that \(f(x_1) = -x_1\) and \(f(x_2) = -x_2\).

We know then based on the assumption that \(x_1 < x_2\) then \(-x_1 > -x_2 \Rightarrow f(x_1) > f(x_2) \)

Thus \(x_1 < x_2 \Rightarrow f(x_1) > f(x_2)\) which means its strictly decreasing.
