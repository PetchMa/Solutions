Polynomials

Problem 1

Prove: that \(f(x)=\sin x\) is not a polynomial.

Proof By Contradiction.

Assume that \(f(x)=\sin x\) is not a polynomial. If \(f(x)\) is a polynomial, then we can write the function as the product of \((x-a_i)\) factors, where \(a_i\) is a root of the function \(f(x)\)

We know the roots for the function \(f(x)\) is

\[\forall a \in \mathbb{Z}, \sin(a\pi) = 0\]

We know then the factors should be...

\[(x-\pi), (x-2\pi), (x-3\pi) ...\]

Thus based on the theroem 3 all polynimals could be written as a product of its factors as such. Lets temporary call this function, \(g(x)\).

\[g(x) = \prod_{i=-\infty}^\infty (x-i\pi)\]

If \(f(x)\) were the same function as \(g(x)\) then \(sin(\frac{\pi}{2}) \) would be a polynomial based on its definition.

We check the value \(x = \frac{\pi}{2}\) for \(f(x)\)

\[f(\frac{\pi}{2}) = \sin(\frac{\pi}{2})= 1\]

We check the value \(x = \frac{\pi}{2}\) for \(g(x)\)

\[g(\frac{\pi}{2}) = \prod_{i=-\infty}^\infty (\frac{\pi}{2}-i\pi)\]

Notice: that \(|\frac{\pi}{2}-i\pi| > 1, \forall i \in \mathbb{Z}\). We know that the product of values greater than 1, will also always be greater than 1. Thus we prove that...

\[g(\frac{\pi}{2}) \neq 1\]

Thus clearly...

\[f(\frac{\pi}{2}) \neq g(\frac{\pi}{2}) \]

Since \(f(x)\) cannot be written as the product of its "factors" we can conclude that \(f(x)\) is not a polynimal since it contradicts the definition of a polynomial.

Problem 2

Prove: that if two cubic polynomials intersect at 4 or more points, then those polyno- mials are equal.

Let \(f(x) = a_0x^3+a_1x^2+a_2x+a_3\)

Let \(g(x) = b_0x^3+b_1x^2+b_2x+b_3\)

To find the intersection of \(f(x), g(x)\) then we solve for values in which \(f(x) = g(x)\) which is when \(f(x)- g(x) = 0\). Let the function \(p(x) = f(x)- g(x) \). The intersections of \(f, g\) are thus the roots of \(p(x)\).

\[p(x) = f(x)- g(x) \] \[p(x) = ( a_0x^3+a_1x^2+a_2x+a_3)- (b_0x^3+b_1x^2+b_2x+b_3) \] \[p(x) = ( a_0-b_0)x^3+(a_1-b_1)x^2+(a_2-b_2)x+(a_3-b_3) \]

We know that \(p(x)\) is a degree 3 polynomial. From the fundamental theorem of algebra a degree 3 polynomial has at most 3 roots.

The only case where p(x) to have more than 3 roots, would be if the constants where all 0's thus \(p(x)\) would have an infinte number of 0's.

If the constants were all zeros it would suggest that \(f(x) = g(x), \forall x \in \mathbb{R}\) thus we've proven that if two cubic polynomials intersect at 4 or more points, then those polyno- mials are equal.
