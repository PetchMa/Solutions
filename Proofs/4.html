Formal Logic

Consider the following

\[\exists M \in \mathbb{R}, \quad N \in \mathbb{N}: \forall x \in \mathbb{R}, x > N \Rightarrow f(x) \leq M\]

Find a function that satisfy the following logic.

We can translate this logic into English which says: There exists, \(M\) (real) and \(N\) (natural) in which for all real x values that are less then \(N\) the function never surpasses the value \(M\). What we're describing here is the following graphically.

Any function that fits under that interval constraint would satisfy the logic. Infact \(f(x) = -x\) would simply do. In this case we can pick \(f(x) = \frac{1}{x}\)

Prove that it satisfies the following logic.

The logic says there Exists a case where this is true. Thus to prove its existence we just need to show a case where this function satisfies the logic. Produce a \(M \) and \(N\).

Let \(M = 1\), \(N = 2\). \(f(x) = \frac{1}{x}\)

We know that for the case \(N =2, f(2) = \frac{1}{2}\) which satisfies \(f(N) \leq M\)

We can prove that the function \(f(x) = \frac{1}{x}\) is always decreasing along the interval \(x > N\) with the following proof.

\(\forall k \in \mathbb{R}, k>0, (\frac{1}{x+k}-\frac{1}{x}) < 0 , \Rightarrow f(x)\) is decreasing.

\[\frac{1}{x+k}-\frac{1}{x} = \frac{x - (x+k)}{x(x+k)} \] \[\frac{1}{x+k}-\frac{1}{x} = \frac{-k}{x(x+k)} \]

We know that \(x > N \) which \(x > 2\) is always positive. We know that \( k>0\) thus it is evident that \(\frac{1}{x+k}-\frac{1}{x} < 0\) which implies that the function is always decreasing. along this interval.

Since the function is decreasing along this interval the maximal value it can take would be the smallest value \(x\) can take on. Which we know is \(< M\). Since there is no value that can surpass this maximal value (as the function is always decreasing) we can conclude that the condition \(f(x) \leq M\) is thus satisfied.

\(\therefore \) We've proven the existence of such values \(M \) and \(N\) that satisfy the conditions. Thus \(f(x)\) satisfies the logic.

What is the negation of the statement?

We move in as we negate each statement.

\[\neg(\exists M \in \mathbb{R}, \quad N \in \mathbb{N}: \forall x \in \mathbb{R}, x > N \Rightarrow f(x) \leq M)\] \[= \forall M \in \mathbb{R}, \quad N \in \mathbb{N} \neg( \forall x \in \mathbb{R}, x > N \Rightarrow f(x) \leq M)\] \[= \forall M \in \mathbb{R}, \quad N \in \mathbb{N}: \exists x \in \mathbb{R}, (x > N) \wedge (f(x) > M)\]

In English this means: There exists a real number \(x\) such that its \(> \) any natural number \(N\) and \(f(x) > \) any real number \(M\)

What is a function that satisfy the negation statement?

We know that it needs to be bigger than any value \(M\) correct? How do we make sure the function is greater than \(M\)? We add a positive number to it. Thus our function could look like \(f(x) = |x| + M\). We know that \(N\in \mathbb{N}\) so \(x\) is always positive. \(f(x) = x + M\)

Prove it.

Proof by contradiction.

Assume that \(f(x) \leq M\) when \(x > N\).

If the above were true then, \(f(x) - M \leq 0 \) when \(x > N\)

\[f(x) - M = (x+m) - M\] \[f(x) - M = x\]

For \(f(x) - M \leq 0 \Rightarrow x < 0 \). Contradiction: since we defined \(x > N\) where \(N \in \mathbb{N}\) by definition, \(N > 0 \Rightarrow x > 0\). Thus it is impossible for \(f(x) \leq M\) when \(x > N\).

Logic Operations

Converse:

The converse of the statement \(A ⇒ B\) is the statement \(B ⇒ A\)

Contrapositive:

The contrapositive of the statement \(A ⇒ B\) is the statement \(\neg B ⇒ \neg A\). We flip and negative.

Implies argument: \(A \Rightarrow B\)

If our hypothesis is TRUE AND the conclusion is TRUE. We should get TRUE.

If our hypothesis is TRUE AND the conclusion is FALSE. We should get False.

If our hypothesis is FALSE AND the conclusion is TRUE. We can't say its actually false because our assumption was already false. Thus we say this is technically true because its not exactly wrong...
