Problem 6

By Peter Ma | Sept 10th 2020

The power set of a set \(X\) is the set consisting of all subsets of \(X\). The power set of \(X\) is usually written \(P(X)\). If \(X\) is a set with \(n\) elements (i.e. with cardinality \(n\)), what is the cardinality of \(P(X)\)?

Solution

\[2^n\]

For each element of the set, there are 2 choices for either it to be included or excluded from the set when creating the powerset. Thus if there is \(n\) elements there should be \(2^n\) total sets.

We can state the cardinality of the power set to be the of combinations of the elements in set \(x\) choosing from \(1 \rightarrow n\). Example

\[|P(\{1,2,3\})| =|\{\{\emptyset\}\}|+ |\{\{1\}, \{2\}, \{3\}\}|+|\{\{1,2\}, \{1,3\}, \{2,3\}\}|+|\{ \{1,2,3\}\}| \] \[={3 \choose 0}+ {3 \choose 1}+ {3 \choose 2}+{3 \choose 3} =8\]

Thus if we wanted to calculate the cardinality of the power set, we can write it as ...

\[|P(x)| = \sum_{k=0}^{n}{n \choose k} \]

We can use the binomial theorem to prove that ...

\[|P(x)| = \sum_{k=0}^{n}{n \choose k} = \sum_{k=0}^{n}{n \choose k} (1)^{n-k}(1)^k = (1+1)^n = 2^n\]

Thus we have that the cardinality is \(2^n\)
