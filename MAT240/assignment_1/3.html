Let \(f:X→Y\) be a map. The map \(g:Y→X\) is called an inverse of \(f\) when it satisfies the properties \[g \circ f = I_x \quad and \quad f \circ g = I_y \] where \(I_z\) means the identity map from \(z\) to itself.

a) Prove that if an inverse of the map \(f\) exists, then it is unique.

Proof

Let \(k\) be another inverse of \(f\) such that \(k \circ f = I_x\)

\[g = I_x \circ g \] \[g = (k \circ f) \circ g \] \[g = k \circ (f \circ g )\] \[g = k \circ I_x\] \[g = k \]

Thus we've proven that the inverse of \(f\) is unique, since there does not exist \(k\) such that it doesn't equal \(g\) meaning the inverse is unique.

b) Give an example of a non-invertible map \(f\).

Example

Example: \(f: X\rightarrow Y, f(x) = x^2 \)

c) Prove that \(f\) is invertible (i.e. it has an inverse) if and only if it is a bijection.

Proof By Contradiction.

Assume that \(f\) is not bijective and has an inverse.

If \(f\) isn't bijective that would imply that either....

  1. \(f\) is injective. For \(g\), not all \(y: y\in Y\) will be mapped to \(x:x\in X\)

  2. \(f\) is surjective. For \(g\), \(\exists y: y\in Y\) that can be mapped to multiple \(x:x\in X\)

If \(f\) is injective that would imply that not all values of the inverse's domain can be mapped back to its codomain. Meaning that the condition of \(g \circ f = I_x \) will be contradicted.

If \(f\) is surjective that would imply that \(\exists y: y\in Y\) that can be mapped to multiple \(x:x\in X\). Thus it will fail to map back the entire image of the original function back to original domain meaning that the condition of \(f \circ g = I_y \) will be contradicted.

If \(f\) is bijective then, every image mapped by \(f\) can also be mapped back to its domain as its a one to one and onto mapping.

By contradiction we've shown that \(f\) is invertible if and only if it is a bijection.

d) Suppose \(f:X→Y\) and \(g:Y→X\) are such that \(g∘f=I_x\). Does it follow that \(f∘g=I_y\)? Why?

Solution

No. This is because the condition \(g∘f=I_x\) doesn't guarantee that every value given by \(g\) can be mapped in the function \(f\). It only guarantees that every value given by \(f\) can be mapped back to the indentity function.

e) Let \(f,g\) be as in part d), with the additional assumption that \(f\) is surjective. Does it follow that \(f∘g=I_y\)? Why?

Solution

No we can't since if \(f\) is surjective it would mean that the image of the inverse cannot map back all values of \(f\)'s domain. Thus this contradicts that \(f∘g=I_y\).
