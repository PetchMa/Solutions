Let \(f:X→Y\) be a map. The map \(g:Y→X\) is called an inverse of \(f\) when it satisfies the properties \[g \circ f = I_x \quad and \quad f \circ g = I_y \] where \(I_z\) means the identity map from \(z\) to itself.

a) Prove that if an inverse of the map \(f\) exists, then it is unique.

Proof

Let \(k\) be an inverse of \(f\) such that \(k \circ f = I_x\)

\[g = I_x \circ g \] \[g = (k \circ f) \circ g \] \[g = k \circ (f \circ g )\] \[g = k \circ I_y\] \[g = k \]

Thus we've proven that the inverse of \(f\) is unique, since there does not exist \(k\) such that it doesn't equal \(g\) meaning the inverse is unique.

b) Give an example of a non-invertible map \(f\).

Example

Example: \(f: X\rightarrow Y, f(x) = x^2 \)

c) Prove that \(f\) is invertible (i.e. it has an inverse) if and only if it is a bijection.

Proof By Contradiction.

Assume that \(f\) is not bijective and has an inverse.

If \(f\) isn't bijective that would imply that either....

  1. \(f\) is injective.

  2. \(f\) is surjective.

  3. \(f\) is neither.

If \(f\) is injective it cannot guarantee that all values of the inverse's domain can be mapped to \(I_x\). Meaning that the condition of \(f \circ g(I_y) = I_y \) will be contradicted.

If \(f\) is surjective that would imply that there could \(\exists x_1,x_2\in X: x_1\neq x_2\) and \(f(x_1)=f(x_2)\) that would imply that \(g\circ f (x_1)\) could equal \(x_1\) or \(x_2\). That contradicts the definition of a mapping where the element of a domain, cannot be mapped to two different images.

If \(f\) is neither then its neither injective nor surjective. This implies there exist multiple elements of the domain which maps to the same image, AND not all elements of the codomain is mapped by \(f\). The fact that not all elements of the codomain of \(f\) has a map already contradicts the condition of \(f \circ g(I_y) = I_y \). Thus \(f\) wouldn't be invertable.

If \(f\) is bijective then by definition, \(\forall x_1,x_2 \in X: x_1\neq x_2, f(x_1)\neq f(x_2)\) AND \(\forall y \in Y, \exists x \in X :f(x)=y\). This would mean that \(g \circ f = I_x\) since every \(y\in Y\) is mapped only once to an \(x \in X\). \(f \circ g = I_y\) would also be guaranteed since by definition of bijection every value \(y\in Y\) is mapped by \(f\) from \(x \in X\) thus \(f \circ g = I_y\).

By contradiction we've shown that \(f\) is invertible if and only if it is a bijection.

d) Suppose \(f:X→Y\) and \(g:Y→X\) are such that \(g∘f=I_x\). Does it follow that \(f∘g=I_y\)? Why?

Solution

No. This is because the condition \(g∘f=I_x\) doesn't guarantee that every value given by \(g\) can be mapped in the function \(f\). There can still exist a value that \(g\) returns that does not get mapped back by \(f\).

e) Let \(f,g\) be as in part d), with the additional assumption that \(f\) is surjective. Does it follow that \(f∘g=I_y\)? Why?

Solution

Yes we can. Firstly, since \(f\) is surjective, by definition every element of \(Y\) has at least one map from set \(X\). But since we know that \(g∘f=I_x\) it would mean every map \(f\) has one map, or else, there could \(\exists x_1,x_2\in X: x_1\neq x_2\) and \(f(x_1)=f(x_2)\) that would imply that \(g\circ f (x_1)\) could equal \(x_1\) or \(x_2\). That contradicts the definition of a mapping.
