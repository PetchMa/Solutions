a) Compute \(\cos(\frac{7}{24}\pi)\)

Solution

We can rewrite \(\frac{7}{24}\pi\) as the following \(\frac{1}{2}(\frac{6}{24}\pi + \frac{8}{24}\pi)\). We can then use the half angle formula for \(\cos a\)

\[ \cos(\frac{7}{24}\pi) =\cos(\frac{1}{2}(\frac{6}{24}\pi + \frac{8}{24}\pi)) \] \[ \cos(\frac{7}{24}\pi) =\sqrt{\frac{1+\cos(\frac{6}{24}\pi + \frac{8}{24}\pi) }{2}} \]

Since \(\frac{7}{24}\pi\) is in quadrant 1 ^, the value is always positive, thus we take only the positive result.

Compound angle identity.

\[ \cos(\frac{7}{24}\pi) = \sqrt{\frac{1+\cos(\frac{6}{24}\pi)\cos(\frac{8}{24}\pi) - \sin(\frac{6}{24}\pi)\sin(\frac{8}{24}\pi) }{2}} \]

Simplify...

\[ \cos(\frac{7}{24}\pi) = \sqrt{\frac{1+\sqrt{2}/4 - \sqrt{6}/4 }{2}} \] \[ \cos(\frac{7}{24}\pi) = \sqrt{\frac{4+\sqrt{2} - \sqrt{6} }{8}} \]

b) Express \(\sin(5α)\) in terms of \(\sin(α)\).

Solution

We can rewrite the following \(\sin(5α)\) as \(\sin(2\alpha+(2\alpha+\alpha))\)

\[\sin(5α) = \sin(2\alpha+(2\alpha+\alpha))\]

Compound angle identity.

\[\sin(5α) = \sin(2\alpha)\cos(2\alpha+\alpha) + \cos(2\alpha)\sin(2\alpha+\alpha)\]

Apply the compound angle identity again.

\[\sin(5α) = \sin(2\alpha)[\cos(2\alpha)\cos(\alpha) - \sin(2\alpha)\sin(\alpha)] + \cos(2\alpha)[\sin(2\alpha)\cos(\alpha) + \cos(2\alpha)\sin(\alpha)]\]

Use the double angle identities.

\(\sin(5α) = 2\sin(\alpha)\cos(\alpha)[ (\cos^2(\alpha)-sin^2(\alpha))\cos(\alpha) - 2\sin(\alpha)\cos(\alpha)\sin(\alpha)] + (\cos^2(\alpha)-sin^2(\alpha))[2\sin(\alpha)\cos(\alpha)\cos(\alpha) + (\cos^2(\alpha)-sin^2(\alpha))\sin(\alpha)]\)

Collect like terms and distribute in.

\(\sin(5α) = 2\sin(\alpha)\cos(\alpha)[ (\cos^2(\alpha)-sin^2(\alpha))\cos(\alpha) - 2\cos(\alpha)\sin^2(\alpha)] + (\cos^2(\alpha)-sin^2(\alpha))[2\sin(\alpha)\cos^2(\alpha) + (\cos^2(\alpha)-sin^2(\alpha))\sin(\alpha)]\)

\(\sin(5α) = [ (\cos^2(\alpha)-sin^2(\alpha))2\cos^2(\alpha)\sin(\alpha) - 4\cos^2(\alpha)\sin^3(\alpha)] + (\cos^2(\alpha)-sin^2(\alpha))[2\sin(\alpha)\cos^2(\alpha) + (\cos^2(\alpha)-sin^2(\alpha))\sin(\alpha)]\)

\(\sin(5α) = [ 2\cos^4(\alpha)\sin(\alpha)- 2\cos(\alpha)\sin^3(\alpha)-4\cos^2(\alpha)\sin^3(a)] + (2\cos^4(\alpha)\sin(\alpha) + \cos(\alpha)\sin(\alpha) - \cos^2(\alpha)\sin^3(\alpha)+ (-2\sin^3(\alpha)\cos^2(\alpha) - \cos^2(\alpha)\sin^3(\alpha) + \sin^5(\alpha)) \)

Collecting all the like terms we get the following...

\[\sin5\alpha=5\cos^4(\alpha)\sin(\alpha)−10\cos^2(\alpha)\sin^3(\alpha)+\sin^5(\alpha)\]

We know that \(\cos^2 x = 1-\sin^2 x\) thus we can rewrite with just \(\sin \alpha\)

\[\sin5\alpha=16\sin^5\alpha−20\sin^3\alpha+5\sin \alpha\]
