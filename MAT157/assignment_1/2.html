a) Prove that \(\sqrt{273}\) is irrational.

Proof By Contradiction

Assume \(\sqrt{273}\) is rational which implies that we can write it as \(\sqrt{273} = \frac{p}{q}: p,q \in \mathbb{Z}\)

\[\sqrt{273} = \frac{p}{q}\] \[q\sqrt{273} = p\] \[273\cdot q^2 = p^2\]

We know that \(p,q\) can be uniquely prime factorized. We know that \(273\) can be prime factorized into \(3 \cdot 7 \cdot 13\)

We can write \(p^2\) as the prime factor of \(p\) except each factor appears twice. This implies that it MUST have an even number of factors.

We can write \(q^2\) as the prime factor of \(q\) except each factor appears twice. This implies that it MUST have an even number of factors.

Contradiction. \(p^2\) has an even number of prime factors, however \(q^2 \cdot (3 \cdot 7 \cdot 13)\) clearly has an odd number of factors. The equality is false, thus we cannot write \sqrt{273} in the form \(\frac{p}{q}, \quad p,q \in \mathbb{Z}\), this contradicts the premise that \(\sqrt{273}\) is rational. Thus it must be irrational.

b) Prove that \(\sqrt[3]{11}\) is irrational.

Proof By Contradiction

Assume \(\sqrt[3]{11}\) is rational which implies that we can write it as \(\sqrt[3]{11} = \frac{p}{q}: p,q \in \mathbb{Z}\) also there are no common factors between \(p,q\). [Relatively prime].

\[\sqrt[3]{11} = \frac{p}{q}\] \[q\sqrt[3]{11} = p\] \[11\cdot q^3 = p^3\]

This implies that \(11\) is a factor of \(p^3\), and thus a factor of \(p\) since its the cube of \(p\). We can write it as such then \(p = 11n : n\in \mathbb{Z}\). Lets replace it for \(p\)

\[q\sqrt[3]{11} = 11n\] \[11\cdot q^3 = 11^3p^3\] \[ q^3 = 11^2p^3\]

This would imply that \(11\) is ALSO a factor of \(q\). This contradicts the premise that \(p,q\) are relatively prime. Thus we can not write \(\sqrt[3]{11}\) as a rational number. Thus it must be irrational.

c) Prove that \(\sqrt{4 + 2\sqrt{2}} - \sqrt{3}\) is rational.

We can replace a few of the square roots with trig values. Like such.

\[=\sqrt{4 + 4\cos(\frac{\pi}{6})} - 2\cos(\frac{\pi}{6})\]

Factor.

\[= 2\sqrt{1 + 1\cos(\frac{\pi}{6})} - 2\cos(\frac{\pi}{6})\]

We can rewrite the square root into the form of a half angle formula.

\[=2\sqrt{\frac{1 + 1\cos(\frac{\pi}{6})}{2}\cdot 2} - 2\cos(\frac{\pi}{6})\]

Replace with half angle.

\[=2\sqrt{2}\cos(\frac{\pi}{12})- 2\cos(\frac{\pi}{6})\]

We also want to write it such that the angles are the same and thus we can replace with the double angle formula.

\[=2\sqrt{2}\cos(\frac{\pi}{12})- 2(2\cos^2(\frac{\pi}{12})-1)\]

We can compute the trig values by decomposing the angles as such...

\[=2\sqrt{2}\cos(\frac{4\pi}{12}-\frac{3\pi}{12})- 2(2\cos^2(\frac{4\pi}{12} - \frac{3\pi}{12})-1)\]

Simplifies to the following...

\[=2\sqrt{2}\frac{\sqrt{6}+\sqrt{2}}{4} - 4(\frac{\sqrt{6}+\sqrt{2}}{4})^2+2\] \[=\frac{2\sqrt{12}+4}{4} - 4(\frac{6+2\sqrt{12}+2}{16})+2\] \[=\frac{2\sqrt{12}+4}{4} - (\frac{6+2\sqrt{12}+2}{4})+2\] \[=\frac{4}{4} - (\frac{8}{4})+2\] \[=1\]
